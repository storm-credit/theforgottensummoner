# 수술용 메스 세트 (Surgical Scalpel Set)

> *"이 메스는 살리기 위해 쓴다. 하지만 필요하다면— 죽일 수도 있다."*
> — 카라 알라무르

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **분류** | 무기 — 정밀 단검 세트 (Precision Scalpel Set) |
| **등급** | **정밀급 (Precision)** ✦ (비표준 등급 — 유일 인증 의료 무기) |
| **가치** | 500만 골드 (미스릴 재료 + 카라의 맞춤 제작 가치) |
| **현재 소유자** | **[[13. 카라 알라무르 (Kara Al'Amur)]]** |
| **세력 귀속** | [[4. 붉은 사막 부족 (Red Desert Tribes)]] |
| **제작자/장인** | 자유도시 연합 최고급 미스릴 세공 장인 — 카라가 의뢰·설계 |
| **재료** | 최상급 미스릴, 청정 마정석 (오염 방지), 카라의 설계도면 기반 제작 |
| **거래 가능 여부** | 불가 (카라의 의료 마력에 동조된 맞춤 도구) |
| **강화 가능 여부** | 가능 (미스릴 재가공으로) |

### 외형 묘사
크기가 다른 미스릴 메스 7개, 집게 3개, 수술 바늘 한 세트로 구성된다. 모두 까만 가죽 파우치에 정리되어 있으며, 카라는 이것을 허리에 차고 다닌다. 각 메스의 날은 빛이 굴절될 정도로 얇고 예리하며, 사용 후 자동 세척 마법으로 항상 청결을 유지한다. 전투에서 무기로 사용할 때는 급소를 정확히 겨냥한 한 발만으로도 상대를 제압할 수 있다.

## 2. 성능 및 효과 (Stats & Effects)

| 유형 | 효과 | 비고 |
|------|------|------|
| **기본 공격력** | 낮음 (비전투용) | 급소 명중 시 치명 |
| **절삭력** | 세공 가능한 모든 물질 절단 — 미스릴 수준 경도 | 항상 발동 |
| **급소 파악** | 의학 지식 기반, 급소 명중 확률 +80% | 카라 사용 시 |
| **치유 연동** | 힐링 샌드 스킬과 연계 시 상처 봉합 속도 3배 | 치료 중 발동 |
| **속성** | 물리 (순수) | |

### **고유 능력: 《정밀 마취 (Precise Anesthesia)》**
- **발동 조건:** 카라가 메스로 상대방의 정확한 신경 급소를 찌를 때 (호신 또는 치료 목적)
- **효과:** 마취 효과로 대상이 즉시 의식을 잃는다. 죽지 않고 완전히 무력화되므로 아군 부상자를 수술할 때도, 사로잡은 포로를 안전하게 제압할 때도 쓰인다. 카라는 이 능력을 비전투 목적으로만 사용하는 원칙을 지키지만, 위협 상황에서는 예외를 둔다.
- **지속 시간:** 즉발 마취 (지속 30분) / 재사용 대기: 없음

### 보조 능력
- **오염 방지:** 사용 도중 외부 독·세균이 메스에 묻어도 청정 마정석이 즉시 중화
- **독 분석:** 미스릴 날에 닿은 독성 물질의 성분을 카라가 감각으로 분석 가능
- **절단 후 봉합:** 메스로 자른 상처를 수술 바늘로 완벽히 봉합 — 흉터 남지 않음

## 3. 배경 스토리 & 전승 (Lore)

### 제작 기원
카라가 의료 활동 중 저급 도구의 한계를 절감한 후, 수년간 돈을 모아 자유도시 연합의 최고급 장인에게 의뢰해 자신이 직접 설계한 메스 세트를 제작했다. 제작 비용이 너무 비싸 아버지의 빚도 다 갚지 못하고 있다는 사실을 아리안이 나중에 알게 되어 몰래 도와주었다.

### 전승
"카라의 메스에 닿으면 죽지 않는다"는 말은 붉은 사막 부족에서 신뢰의 의미로 쓰인다. 역설적으로 그 날카로운 메스가 수백 명의 목숨을 살렸기 때문이다.
