# 바다의 교단 내부 세력 동역학

> *"우리는 심연 아래 모순을 안고 나아간다. 정화와 자비는 양립할 수 있는가?"*

---

## 1. 4대 파벌 구조

```
           ┌──────────────────────────────────┐
           │        🏛️ 대제사장실 (행정)       │
           │  (님부스 대제사장의 직할 성기사 지휘)    │
           └──────────┬───────────────────────┘
                      │  
 ┌────────────────────┼───────────────────────────┐
 │                    │                           │
 │ ⚔️ 정화청 (이단 심문) ─┼─(견제)── 🕊️ 자비의 물결 총국 │
 │ (내부 감사, 무자비)    │               (구호, 민심수습) │
 │                    │                           │
 ├──────────(정보 제공)─┴──────────────(물자 지원)──┤
 │                                                │
 │                👁️ 심해 감시국                   │
 │        (은둔, 심연 균열 방어의 최전선)            │
 └────────────────────────────────────────────────┘
```

> [!important] 
> 겉으로는 교단의 온화함을 담당하는 **자비의 물결**이 민심과 자본을 움켜쥐고 있으며, 교리의 순수성과 치안은 **정화청**이 강경하게 틀어쥐고 있어 내부 알력이 존재한다. 이 모든 것을 대제사장이 타이드워커 기사단을 통해 조율 중.
