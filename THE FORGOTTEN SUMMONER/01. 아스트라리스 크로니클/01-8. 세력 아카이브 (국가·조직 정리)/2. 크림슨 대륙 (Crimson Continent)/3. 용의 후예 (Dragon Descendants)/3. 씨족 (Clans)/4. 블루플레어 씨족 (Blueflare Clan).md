# 블루플레어 씨족 (Blueflare Clan)

> *"푸른 불꽃은 붉은 불꽃보다 뜨겁다. 지식은 힘보다 오래간다."*

---

## 1. 씨족 개요

| 항목 | 내용 |
|------|------|
| **씨족명** | 블루플레어 씨족 (Blueflare Clan) |
| **지위** | 용의 후예 수석 마법사 · 기술·방어 총괄 |
| **본거지** | [[드래곤포지 (Dragon Forge)]], 해안 요새 |
| **핵심 자산** | 용의 방패단, 마법 연구소, 드래곤포지, 연금술 공방 |
| **정치 노선** | 수호파 핵심 · 전통 보존 · 방어 강화 · 지식 우선 |

> [!important] 용의 후예의 두뇌와 방패
> 라미엘 블루플레어(S급)의 마법, 바란 스톤스케일(S급)의 방어,
> 울프가르 드래곤포지(A급)의 무구 제작이 합쳐져
> 용의 후예가 소수 정예로도 강한 이유를 만든다.
> 이들이 없으면 주전파도 싸울 무기가 없다.

## 4. 내부 갈등

- **라미엘 vs 레가스**: 지식 vs 힘 — 수호파와 주전파의 근본적 대립
- **바란의 고립**: 방어만 하면 영토가 넓어지지 않는다 — 주전파의 비판
- **울프가르의 중립**: 무기를 만드는 자는 쓰는 자를 선택하지 못한다
- **이젤의 해안 위협**: 해상에서 오는 미지의 위험 — 심연? 솔라리안 해군?
- **드락사르의 실험**: 용의 피를 이용한 연금술 실험 — 용의 반발 가능
