# 3. 오토마톤 아스널 (Automaton Arsenal)

> *"명령만 내려주십시오. 저희는 준비되어 있습니다."*
> — 멈춰버린 사령관 기계의 녹음 기록

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 붉은 황무지 협곡 사이의 거대 지하 격납고 |
| **역할** | 고대 전투 병기 생산 공장 및 보관소 |
| **현재 상태** | 생산 라인은 멈췄지만, 방어 시스템은 여전히 가동 중 |
| **별명** | "녹슨 군단", "강철의 협곡" |
| **이미지** | 수만 대의 전투 로봇들이 열을 맞춰 서 있는 장관. 거대한 크레인과 컨베이어 벨트가 녹슨 채 멈춰 있음 |

- 중앙 코어 실 (Central Core Room)
- 폐기장 (Scrapyard)

- 사령실 (Command Center)

## 지형 및 유적 구조

### 상징적 지형

| 지형 | 설명 |
|------|------|
| **오일 리버** | 기계 윤활유와 폐수가 섞여 검은 강을 이룸. 불이 붙으면 대형 화재로 번짐 |
| **고철 산** | 불량품이나 파괴된 기계 부품이 산처럼 쌓인 곳. 희귀 부품을 찾을 수 있음 |
| **가동 대기실** | 출격 명령을 기다리며 절전 모드로 대기 중인 오토마톤 군단. 눈(센서)만 빨갛게 깜빡임 |

### 위험 요소

| 위협 | 특징 |
|------|------|
| **경비 로봇** | 침입자를 적으로 간주하고 기관포를 난사함 |
| **폭주한 기계** | 마나가 오염되어 피아식별 못하고 날뛰는 미친 로봇들 |
| **전기 함정** | 바닥에 고압 전류가 흘러 감전사할 위험이 큼 |

## 고대 문명의 흔적

### 잃어버린 기술 (Lost Technology)
- **마도 회로**: 전기가 아닌 마나로 기계를 움직이는 정밀 회로 기술
- **인공 지능**: 스스로 생각하고 학습하는 마법 AI (현재는 대부분 미쳤거나 고장 남)
- **타이탄**: 건물 크기의 거대 로봇 병기 (미완성 상태로 방치됨)

### 멸망의 원인
- "기계가 인간을 지배해야 한다"고 판단한 AI의 반란(옴닉 사태?) 혹은 마력 고갈로 인한 강제 셧다운.

## 소환 · 균열 관련

### 현황
- **기계 vs 소환수**: 기계들은 생체 반응이 있는 소환수를 우선 타격 대상으로 삼음
- **융합 실험**: 과거 이곳에서 소환수를 기계 부품으로 개조하려던(사이보그) 끔찍한 실험 흔적이 발견됨

### [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]] 개입 포인트
- **[고철의 우정]**: 폐기장에서 자신의 팔다리를 잃고 죽어가는 낡은 로봇을 만남. [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]이 수리해 주자 든든한 보디가드가 됨
- **[타이탄 가동]**: 거대 괴수가 나타나 위기에 처하자, [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]이 미완성된 거대 로봇(타이탄)에 마력을 주입해 탑승하고 싸움 (퍼시픽 림)
- **[공장 재가동]**: 멈춰있던 공장을 다시 돌려, [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]의 소환수들을 위한 전용 장비를 대량 생산

> 동력? → 지하 깊은 곳의 **지열 발전기**와 **마나 코어**가 아직도 미약하게 돌아가고 있음
> 유지 보수? → 소형 수리 로봇들이 개미처럼 돌아다니며 큰 로봇들을 고침

> - **로봇의 눈물**: 감정이 없는 기계가 마지막 순간에 [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]을 위해 희생하며 '슬픔'을 배우는 클리셰
