# 아르카나 잔재 가문 관계도 (Arkana Remnants Political Dynamics)

> *"제국은 무너졌지만, 명령 체계는 여전히 황제의 한 마디에 움직인다."*

---

```
            ┌──────────────────────────────────────┐
            │  아르카노스 황실 (마법 황제 · 절대 권력)  │
```

> [!important] 멸망한 제국의 부활
> 아르카나 잔재는 300년 전 멸망했다고 알려졌으나,
> 황제 제레시스가 수도 전체를 **이공간으로 대피**시켰다가 최근 귀환했다.
> 내부적으로 **황제(제레시스)의 극단적 노선 vs 황태자(크세노스)의 온건 노선**이 충돌.

## 2. 세력 간 관계 정리

| 관계 | 유형 | 설명 |
|------|------|------|
| **황실 ↔ 벨로그** | 동맹/위험 | 네크로스의 강령술이 군사력의 핵심 — 하지만 황제도 두려워하는 힘 |
| **황실 ↔ 이그나토스** | 전투 도구 | 발리안의 화염과 아스모델의 폭풍이 전장의 핵심 전력 |
| **황실 ↔ 파브리** | 기술 의존 | 테크노스의 마도 공학이 오토마톤 군단의 유일한 공급원 |
| **벨로그 ↔ 이그나토스** | 마찰 | 강령술 vs 파괴 마법 — 전쟁 수행 방식의 근본적 대립 |
| **벨로그 ↔ 파브리** | 혐오 | 생체 실험(세바스티안) vs 기계 공학(테크노스) — 방법론 충돌 |
| **이그나토스 ↔ 파브리** | 실무 협력 | 화염 무기 + 마도 공학의 조합 — 전장 병기 공동 개발 |
| **제국본대 ↔ 보이드** | 격리/기피 | 보이드 엠브레이서는 300년 이공간 칩거의 부작용으로 탄생한 괴물들. 통제가 안 되어 일식의 차원 틈새에 격리 중이나, 결전 병기로 투입 준비 중 |

## 3. 4대 세력의 정치적 입장

### 아르카노스 황실

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 아르카나 제국 부활, 솔라리안 제국 복수, 대륙 패권 탈환 |
| **마법 노선** | 모든 마법은 도구 — 금지된 것은 없다 |
| **내부 약점** | 황제-황태자 노선 대립이 사실상 내전의 씨앗 |

> [!warning] 황제 vs 황태자
> 제레시스(SSS급)는 **"지상의 생명력을 흡수해서라도 제국을 재건한다"**는 극단파.
> 크세노스(SS급)는 **"착취하면 과거의 실수를 반복한다"**며 비밀리에 반발.
> 크세노스가 에반 일행과 접촉하여 거래를 제안하는 서사가 핵심 전개점.

### 벨로그 가문 (강령·심연)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 죽음을 초월한 전력 확보, 생사의 경계 연구 |
| **정치적 위치** | 검은 서고회(비밀 결사) 수장 — 외부 노출 금지 |
| **내부 약점** | 금지 연구가 다른 파벌의 반발과 두려움을 유발 |

### 이그나토스 가문 (파괴·폭풍)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 화염과 폭풍으로 적진 초토화, 전장 지배 |
| **정치적 위치** | 루인 비전단 + 스톰 콜러 운용 — 제국의 공격력 핵심 |
| **내부 약점** | 파괴만 알고 건설을 모름 — 전후 통치 능력 부재 |

### 파브리 가문 (마도공학)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 골렘·오토마톤·마도 병기 생산, 기술 우위 확보 |
| **정치적 위치** | 마도 공학부 총괄 — 자원 부족이 최대 병목 |
| **내부 약점** | 원재료(광석·마나석) 부족으로 생산 한계 |

## 4. 균형 유지 메커니즘

### 마법 결투(Arcane Duel) 제도
- 상위 계급에 도전하려면 **마법 결투에서 승리**해야 함
- 패자는 계급 강등 또는 추방 — "실력이 곧 정의"인 야만적 실력주의
- 이 제도가 파벌 간 무력 균형을 유지하는 동시에 내부 소모를 야기

### 잿더미 의회의 형식적 합의
- 황제 복귀 후 권한 축소됐지만, 온건파의 목소리가 이곳에서 유일하게 표출
- 크세노스가 이 의회를 통해 극단 노선을 견제하려 시도

### 자원의 병목
- 오토마톤과 마도 병기 모두 **마나석·광석**이 필수
- 이 자원을 통제하는 자(현재 테크노스)가 실질적인 2인자

## 5. 위기 시나리오

### 시나리오 A: 황태자의 반란

> *"아버지의 길은 파멸이다. 나는 다른 길을 택한다."*

- **촉발**: 크세노스가 에반 일행과 비밀 동맹 체결
- **전개**: 차원 연구소 + 레거시 추적단 vs 황제 직속 비전 기사단
- **위험**: 내전 중 솔라리안 제국이 총공세 — 양면전쟁
- **결과**: 성공 시 온건파 정권 수립, 실패 시 크세노스 숙청

### 시나리오 B: 강령술의 폭주

> *"죽은 자는 명령에 복종한다. 하지만 심연의 존재는 아무도 복종하지 않는다."*

- **촉발**: 네크로스의 강령 실험이 차원 균열을 확대 → 심연의 괴물 대량 출현
- **전개**: 모든 파벌이 생존을 위해 일시 협력 — 하지만 계속 불신
- **위험**: 심연의 존재가 통제 불능이면 아르카나 잔재 자체가 멸망
- **결과**: 성국의 성전 선포 명분 제공

### 시나리오 C: 자원 고갈

> *"마나석이 바닥나면, 골렘은 녹슬고, 군대는 해산된다."*

- **촉발**: 솔라리안 제국이 마나석 공급로 완전 차단
- **전개**: 파브리 가문 고립 → 오토마톤 정지 → 전선 붕괴
- **위험**: 자원 확보를 위한 극단적 약탈·지맥 흡수 가속
- **결과**: 대륙 전체의 마나 순환 교란 → 이상 기후·괴물 활성화

## 6. 서사 활용 포인트

> [!tip] 스토리 핵심
> - **핵심 갈등**: 제국 부활의 방법론 — 착취 vs 공존
> - 크세노스의 비밀 접촉이 에반 서사의 핵심 전개점
> - 솔라리안과의 전쟁이 크림슨 대륙 전체를 불안정하게 만듦
> - 강령술·차원 연구가 세계관 전체의 위협 요소
> - "악역이지만 사정이 있는" 세력 — 300년 동안 이공간에 갇혀 있던 자들
