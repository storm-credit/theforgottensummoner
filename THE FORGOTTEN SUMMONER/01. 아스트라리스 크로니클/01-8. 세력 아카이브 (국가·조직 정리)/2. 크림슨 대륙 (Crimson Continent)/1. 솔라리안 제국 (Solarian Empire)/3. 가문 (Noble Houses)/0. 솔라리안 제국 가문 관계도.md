> [!important] 전제군주제의 핵심
> 솔라리안 제국에서 **모든 권력은 황제에게서 나온다.**
> 귀족은 혈통보다 **전쟁터의 무훈**으로 서열이 결정되며,
> 황제의 신임을 잃으면 작위 박탈과 유배가 즉시 실행된다.
> "피보다 충성, 충성보다 승리" — 이것이 제국의 교리.

## 2. 세력 간 관계 정리

| 관계 | 유형 | 설명 |
|------|------|------|
| **황실 ↔ 스틸그레이트** | 칼과 방패 | 발터는 황제의 가장 신뢰하는 장군 — 동시에 가장 강력한 견제 대상 |
| **황실 ↔ 솔라페인** | 도구 | 마법과 기술을 제국의 도구로 활용 — 어빈의 자율권을 최소화 |
| **황실 ↔ 택티쿠스** | 눈과 귀 | 엘라리안의 첩보가 황제의 안전을 보장 — 하지만 너무 많이 안다 |
| **스틸그레이트 ↔ 솔라페인** | 마찰 | 전통 군부 vs 마도 병기 — 기술이 군대를 대체할지의 갈등 |
| **스틸그레이트 ↔ 택티쿠스** | 견제 | 군부를 감시하는 첩보 vs 감시당하기 싫은 군부 |
| **솔라페인 ↔ 택티쿠스** | 실무 협력 | 첩보와 기술의 조합 — 하지만 비밀 연구에 대한 감시 문제 |
| **제국(4권력) ↔ 이클립스**| 항시 대립 | 제국은 신디케이트를 오물 취급하며 가이우스(친위)가 소탕, 택티쿠스는 침투, 신디케이트는 멸망을 주도 |

## 3. 4대 세력의 정치적 입장

### 솔라리안 황실

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 제국의 절대 통치, 태양신 신앙 기반 정당성 유지 |
| **군사 노선** | 확장주의 — 주변 세력을 정복 또는 동화 |
| **내부 약점** | 황제 1인 독재의 병목 — 레가시우스가 쓰러지면 즉시 계승 전쟁 |

> [!warning] SSS급 황제의 그림자
> [[01-14. 영웅 백과 (Hero Archive)/01-14-2. 현존 영웅/2. 크림슨 대륙 (Crimson Continent)/1. 솔라리안 제국 (Solarian Empire)/01. 레가시우스 솔라리안 (Legasius Solarian)]]은 **SSS급 전사이자 절대군주**다.
> 그의 존재 자체가 제국의 억지력이지만, 동시에 그가 없는 제국은
> 내부 파벌 간 쿠데타로 24시간 내 붕괴할 수 있다.

### 스틸그레이트 가문 (군부)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 전통적 군사력의 유지와 확장, 정복 전쟁 주도 |
| **정치적 위치** | 제국군의 실질적 지휘권 — 발터+칼리스가 양대 군단장 |
| **내부 약점** | 마도 병기가 전통 군을 대체하면 존재 이유 상실 |

### 솔라페인 가문 (마법·기술)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 마법과 기계 기술(마도공학)의 발전, 제국의 기술적 우위 확보 |
| **정치적 위치** | 제국 내 유일한 공인 마법 연구 기관 — 마법 감시국과 연계 |
| **내부 약점** | 군부가 마도 병기의 독립적 발전을 경계하여 예산 견제 |

### 택티쿠스 가문 (첩보)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 내부 반란 사전 차단, 외부 첩보, 정보 우위 |
| **정치적 위치** | 황제의 눈과 귀 — 블랙로즈 군단과 나이트쉐이드 척탄병 운용 |
| **내부 약점** | 너무 많은 비밀을 아는 것의 위험 — 황제도 경계 |

## 4. 균형 유지 메커니즘

> *"태양은 모든 것을 비추지만, 하나만 빛나면 그것은 태양이 아니다."*

### 황제의 분할 통치

- 군부(스틸그레이트)가 독주하면 → **첩보(택티쿠스)를 동원하여 내부 감시**
- 마법(솔라페인)이 위험한 연구 시도 → **이단심문소(루나)가 금지 마법 단속**
- 첩보(택티쿠스)가 과잉 권력 → **군부(스틸그레이트)에게 직접 견제 명령**

### 전쟁평의회의 균형

- 군단장(발터·칼리스) + 첩보(엘라리안) + 마법(어빈) + 황제 = 4자 회의
- 어느 한 쪽이 독주하면 나머지 3자가 자동으로 견제하는 구조

### 이단심문소의 비상 권한

- 루나 선워커의 이단심문소가 **모든 가문을 초월하는 비상 수사권** 보유
- 금지 마법·반역 혐의 시 → 귀족이라도 체포·심문 가능

## 5. 위기 시나리오

### 시나리오 A: 황제 시해 / 무력화

> *"태양이 지면, 별들은 서로를 삼킨다."*

- **촉발**: 레가시우스의 전사 또는 암살 — 아르카나 잔재의 공작 가능성
- **전개**: 발터(군부) vs 엘라리안(첩보) vs 어빈(마법) — 계승 3파전
- **위험**: 제국 군단이 내전으로 빠지면 아르카나 잔재가 즉시 침공
- **외부 개입**: 붉은 사막 부족이 국경 약탈 강화, 용의 후예가 스톰홀드 공격

### 시나리오 B: 마도 병기의 반란

> *"우리가 만든 무기가 우리를 향한다면?"*

- **촉발**: 솔라페인의 마도 공학 실험 폭주 → 오토마톤 or 병기가 통제 불능
- **전개**: 군부가 마법 연구 전면 금지를 주장, 솔라페인 숙청 시도
- **위험**: 마법 기술 없는 제국군이 아르카나의 마법 공세에 무방비
- **외부 개입**: 아르카나 잔재가 폭주한 마도 병기를 포획·역이용

### 시나리오 C: 첩보의 쿠데타

> *"가장 위험한 적은 이미 궁 안에 있다."*

- **촉발**: 엘라리안 택티쿠스가 누적된 비밀 정보를 바탕으로 궁정 쿠데타 시도
- **전개**: 블랙로즈 군단이 황궁 침투, 가이우스의 에테르날 [[01-20. 직업 & 클래스 (Classes & Jobs)/직업 도감/06. 기사 (Knight)]]단과 대치
- **위험**: 성공하면 첩보 국가화, 실패하면 첩보 조직 전면 숙청 → 정보 공백
- **외부 개입**: 혼란을 틈타 모든 적대 세력이 동시 공세
