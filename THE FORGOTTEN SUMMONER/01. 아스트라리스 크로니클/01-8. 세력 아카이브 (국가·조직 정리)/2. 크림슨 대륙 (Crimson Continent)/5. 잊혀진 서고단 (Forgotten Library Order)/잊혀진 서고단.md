# 잊혀진 서고단 (Forgotten Library Order)

> *"우리는 전쟁을 이기러 온 게 아니다. 다만, **누가 어떻게 싸웠는지** 기록하러 왔을 뿐이다."*

---

## 1. 개요

**지식의 수호자**를 자처하는 비밀 조직. 아르카나 제국 시절부터 이어져 온 고대 도서관을 관리하며, 세계의 위험한 지식을 **보관하거나 파괴**하는 역할을 맡는다.

---

## 2. 문화

### ■ 일상 분위기
- 서고 안: **종이 넘기는 소리, 깃펜 긁는 소리, 속삭이는 토론**
- **기록 고행**: 중요한 역사적 현장에서 식사조차 거르고 기록에 몰두
- 대부분 "지금 일어나는 일"보다 "이미 일어난 일"에 더 관심

### ■ 대표 의식/행사

| 의식 | 설명 |
|------|------|
| **연대기 봉인식** | 기록 완성 시 서고 깊은 곳에서 봉인 주문과 함께 묻음 |
| **기억자의 장례** | 화장 후 재 일부를 잉크에 섞어 "마지막 기록" 남김 |

### ■ 서사 포인트
- 말투가 전반적으로 **과거형**: "이 전쟁은 과거 어느 왕조와 닮았습니다"
- 에반에게 **"넌 지금 기록될 이야기의 주인공"** 메타 자각 건드리는 세력

---

## 3. 정치·조직

### ■ 지휘 체계

| 직위 | 역할 |
|------|------|
| **대서고장 (Grand Archivist)** | 최고 책임자, "정보전의 최종 보스" |
| **편년의회 (Chronicle Council)** | 분야별 책임자들 (전장/균열/정치/금단 지식) |
| **현장 관찰관 책임자** | 작전·잠입·탈출 지휘, 가장 "군인에 가까운" |

### ■ 결정 사항
- 어떤 사건을 **어느 관점에서 기록**할지 토론
- 현장팀 직접 파견 vs 외부 의뢰·하청 결정
- 어느 전쟁에 **직접 개입 / 기록만 / 아예 누락**할지 최종 결정

### ■ 조직 철학
> "기록되지 않은 승리와 패배는, 언젠가 다시 반복된다."

> 특이하게도, 전투에서 활약하면 **"학자답지 못하다"**며 오히려 경계 받는 문화.

## 5. 군사 (작전 운용)

> "우리는 싸우러 온 게 아니다. 다만, 싸움이 어떤 의미였는지를 남기러 왔을 뿐."

### ■ 조직 단위

| 부대 | 역할 |
|------|------|
| **현장 서기관 (Field Scribe Unit)** | 3~5인 소규모 팀, 목격+생존+귀환 최우선 |
| **연대기 수호자 (Chronicle Wardens)** | 정예 전투 인력, 서고·유물 수호 |
| **침묵의 사서 (Silent Custodians)** | 암살·기억 조정·비밀 작전 전담 |
| **기록 골렘 (Script Golems)** | 이동식 카메라 + 블랙박스 |

### ■ 전술 기조

1. **정면전 회피, 관찰 우선**: 전선 한 칸 뒤에서 움직임
2. **우선순위**: 기록 > 생존 > 개입
3. **봉인·차단 중심**: 적 죽이기보다 위험 기록/유물 먼저 확보
4. **중립 유지, 다중 계약**: 여러 세력에 다른 버전 판매

### ■ 거점

| 거점 | 특징 |
|------|------|
| **중앙 서고 (Grand Archive Vault)** | 본거지, 위치 불명, 백업 이주 시스템 |
| **이동식 서고 (Moving Scriptorium)** | 대상단으로 위장, 미니 본부 |
| **위성 서고 (Hidden Annexes)** | 각 도시에 위장된 비밀 문서고 |

## 7. 서사 포인트

### ■ "기록을 사러 온 자 vs 기록을 지우러 온 자"

한 사건을 두고:
| 세력 | 요구 |
|------|------|
| 성국 | "이단 탄압의 영광스러운 승리로 기록해 달라" |
| 제국 | "군사적 실책 빼고 균열 대응만 강조" |
| 정령 연합 | "자연 파괴 부분도 반드시 넣어라" |

### ■ 에반 연계

| 포인트 | 내용 |
|--------|------|
| **"미완의 장"** | 서고단은 에반을 연재 중인 연대기로 취급 |
| **서술권 거래** | "네 이야기를 내일의 왕국에게 파는 이야기로 넘겨라" |
| **기억 복원** | 에반의 사이코메트리 + 서고단 기록 마법 = 지워진 진실 복원 |

### ■ "치명적 오발송" 딜레마
- 절대 중립을 천명하는 서고단이 의도치 않게(혹은 침묵의 사서의 공작으로) 제국의 흑역사가 적힌 치명적인 연대기를 '아르카나 잔재'나 '자유도시연합' 측에 떨어뜨리는 사건 발생.
- 에반이 이 거대한 "역사 유출 스캔들"에 휘말려 제국과 서고단 사이에서 줄타기 협상을 벌이게 됨.
