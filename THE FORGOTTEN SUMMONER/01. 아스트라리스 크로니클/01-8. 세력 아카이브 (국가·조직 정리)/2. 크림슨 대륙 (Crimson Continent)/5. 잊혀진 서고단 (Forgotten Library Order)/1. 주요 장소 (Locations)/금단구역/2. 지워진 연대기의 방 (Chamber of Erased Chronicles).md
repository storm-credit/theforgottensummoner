# 2. 지워진 연대기의 방 (Chamber of Erased Chronicles)

> *"어떤 역사는 기록되지 않았다. 아니, 기록되었다가 지워졌다. 우리가 지웠다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 시간의 서고 최심부, 위치 비공개 |
| **분류** | **금단 구역** – 대서고장 직할, 존재 자체 비밀 |
| **역할** | 의도적으로 삭제된 역사 기록 보관 (또는 영구 봉인) |
| **별칭** | "침묵의 서가", "망각의 방", "없는 역사" |

---

## 2. 존재 이유

### 서고단의 딜레마
> "기록되지 않은 것은 반복된다. 하지만 기록되어서는 안 되는 것도 있다."

- 모든 것을 기록하는 것이 서고단의 사명
- 하지만 일부 진실은 **알려지면 더 큰 재앙**
- 해결책: 기록하되, **접근 불가능하게 봉인**

### 삭제 대상
| 유형 | 예시 |
|------|------|
| **세력 요청** | 왕조의 치욕, 전쟁 범죄 증거 |
| **서고단 결정** | 알려지면 세계 불안정 초래 |
| **자발적 봉인** | 너무 위험한 마법/의식 기록 |
| **협박** | 공개 시 서고단 위험 |

---

## 3. 구조 & 보안

### 물리적 구조
| 구역 | 특징 |
|------|------|
| **접근 통로** | **존재하지 않음** – 마법으로만 진입 |
| **외부 결계** | 다중 봉인, 대서고장 열쇠 필요 |
| **내부 공간** | 무한히 확장되는 서가 |
| **보관 방식** | 각 기록이 개별 봉인 상자 |

### 보안 체계
| 층 | 방어 |
|------|------|
| **1층** | 존재 자체 비밀 |
| **2층** | 위치 변경 (정기적 이전) |
| **3층** | 마법 결계 (침묵의 사서 전담) |
| **4층** | 기억 소거 (진입자 기억 조작) |

---

## 4. 보관된 것들

### 삭제된 역사
| 기록 | 내용 |
|------|------|
| **잊혀진 왕조** | 공식 역사에서 완전히 삭제된 왕국 |
| **대학살 진실** | 어떤 전쟁의 진짜 희생자 수 |
| **계약의 배반** | 용의 후예-아르카나 비밀 조약 |
| **심연 실험** | 아르카나 멸망 진짜 원인 |

### "절대 삭제"
- 읽는 것만으로 **재앙을 부르는** 기록
- 파괴 불가능, 영구 봉인만 가능
- 방 안에서도 **별도 격리**

---

## 5. 서고단 내부 갈등

### 보존파
> "우리가 지우면, 누가 진실을 기억하는가?"
- 삭제 최소화 주장, 진실은 언젠가 알려져야 함

### 침묵파
> "알아서는 안 되는 것이 있다."
- 적극적 삭제 지지, 세계 안정이 진실보다 중요

### 균형
- 대서고장이 최종 결정 (동점이면 **보존**)

---

## 6. 소환 & 에반 관련

### 삭제된 진실
- 이 방에는 **공식 역사가 숨긴 모든 것**
- 에반의 과거, 소환사의 역사도 있을 수 있음

### 에반의 사이코메트리
- 봉인 상자에서: 삭제 결정의 무게, 기록자의 고뇌
- 기록 자체에서: **측정 시 위험** – 일부 기록은 역공격
- 방 전체에서: 침묵, 망각, 억압된 진실의 울림

### 에반 서사 포인트
1. **자신의 기록** – 에반에 대한 삭제된 기록 발견
2. **소환사의 진실** – 과거 소환사들의 숨겨진 역사
3. **딜레마** – 알게 된 진실을 공개할 것인가

---

## 7. 스토리 훅

1. **봉인 약화** – 삭제된 기록 중 하나가 "되살아남"
2. **내부 배신** – 누군가 삭제 기록 유출
3. **에반의 접근** – 서고단이 에반의 능력으로 특정 기록 확인 의뢰
4. **진실의 무게** – 알게 된 진실로 에반 도덕적 딜레마
