# 1. 이동식 서고 (Moving Scriptorium)

> *"우리는 어디에도 있고, 어디에도 없다. 역사를 기록하기 위해 역사 속으로 들어간다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 고정 위치 없음, 크림슨 대륙 전역 이동 |
| **역할** | 현장 수집 본부, 위장 거점, 이동식 아카이브 |
| **특징** | 대상단(카라반)으로 위장한 서고단 전진 기지 |
| **별칭** | "움직이는 책", "먼지 상단", "기록자의 집" |

---

## 2. 구조 & 위장

### 외부 위장 (대상단)
| 요소 | 위장 |
|------|------|
| **마차** | 상인 화물차로 위장 |
| **인원** | 상인, 호위병, 하인 역할 |
| **화물** | 겉: 직물, 향신료 / 속: 문서, 장비 |
| **경로** | 교역로를 따라 자연스럽게 이동 |

### 내부 구조
| 마차 | 기능 |
|------|------|
| **본부 마차** | 지휘관 거처, 회의실 |
| **기록 마차** | 현장 수집 문서 정리 |
| **보관 마차** | 중요 문서 보호 (마법 봉인) |
| **호위 마차** | 연대기 수호자 대기 |
| **장비 마차** | 필기구, 복제 장비, 기록 골렘 |

### 보안
- 외부에서 보면 **평범한 대상단**
- 마차 내부는 **마법으로 확장**
- 위급 시 문서 자동 소각 시스템

## 3. 운영

### 편성
| 인원 | 역할 |
|------|------|
| **서기장** | 이동 서고 책임자 |
| **현장 서기관** | 5-10명, 기록 실무 |
| **연대기 수호자** | 5명, 전투/호위 |
| **기록 골렘** | 2-3기, 자동 기록 |
| **위장 인력** | 10명, 상인/마부 역할 |

### 이동 패턴
| 상황 | 이동 |
|------|------|
| **평시** | 교역로 따라 천천히 이동 |
| **사건 발생** | 신속히 해당 지역 접근 |
| **위험 시** | 즉시 철수, 다른 경로 |
| **정기 귀환** | 연 2회 본부(시간의 서고) 복귀 |

### 수집 활동
1. 전쟁/사건 현장 인근 진입
2. 목격자 인터뷰, 현장 기록
3. 물증 수집 (문서, 유물)
4. 정리 후 본부 전달

## 4. 위장 유지

### 거래 활동
- 실제로 **간단한 교역** 수행
- 정보 = 대가, 물건도 거래
- "진짜 상인 같다"

### 발각 위험
| 위험 | 대응 |
|------|------|
| **세관 검사** | 이중 바닥, 마법 은폐 |
| **도적 습격** | 연대기 수호자 대응 |
| **첩자 침투** | 기억 조정 마법 |
| **완전 노출** | 문서 소각 + 분산 철수 |

## 7. 순리 노트 (Plausibility Note)
> [!note] 순리 (Plausibility)
> - **[위장]:** 단순한 상단으로 위장하지 않으면, 값비싼 기록물이나 고대 서적을 노리는 도적들의 표적이 되기 십상임.
> - **[이동성]:** 전쟁이나 재해 현장은 빠르게 변하므로, 고정된 건물이 아닌 이동식 본부가 정보수집에 훨씬 유리함.
> - **[자금]:** 희귀 서적 복제본을 판매하거나, 대상단 행세를 하며 특산품 무역으로 운영 자금을 마련함.

> [!important] 서사 활용 및 비밀
> - **[비밀]:** 서기장의 마차 밑바닥에는 '절대 기록되어서는 안 될 역사'를 적은 검은 노트가 숨겨져 있다.
> - **[위기]:** 제국 검문소에서 마법 탐지기에 걸려 정체가 탄로 날 위기에 처함 ([[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]의 도움으로 모면).
