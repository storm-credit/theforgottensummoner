# 1. 시간의 서고 (Chronos Library)

> *"여기서 시간은 흐르지 않는다. 다만, 쌓이고, 겹치고, 다시 읽힐 뿐이다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 크림슨 대륙 북부 유적대 중심부, 고대 도시 심장부 위 |
| **역할** | 잊혀진 서고단 수도이자 본부 |
| **특징** | 도시 크기의 도서관·아카이브 복합체 |
| **이미지** | 끝없이 이어지는 서가와 탑, 하늘이 보이지 않을 만큼 책으로 가득 찬 석조 도시 |

---

## 2. 주요 구역

| 구역 | 특징 |
|------|------|
| **외곽 연구지구** | 발굴 현장, 야외 연구 텐트, 유물 임시 보관소 |
| **중앙 도서 도시** | 거리 = 서가, 탑 = 전용 아카이브, 광장 = 열람 공간 |
| **심층 아카이브 (Deep Stacks)** | 지하 수십~수백 층, 봉인지식·위험 문서 |
| **시간의 방 (Time Nexus Hall)** | 시간석 중심, 에테르 잔향 재생 시설 |

---

## 3. 조직 구조

| 직책 | 역할 |
|------|------|
| **대서고장** | 서고단 전체 통괄, 외부에 거의 안 보임 |
| **서가장들** | 분야/시대/대륙별 담당 상층 간부 |
| **현장 기록관** | 유적 탐사, 전쟁 기록, 소환 사고 조사 오퍼레이터 |

### ■ 운영 철학
> "진실은 기록되어야 하며, 기록된 진실은 언젠가 다시 읽힌다."

### ■ 비밀 등급
| 등급 | 설명 |
|------|------|
| 개방 | 일반 열람 가능 |
| 제한 | 서가장 허가 필요 |
| 봉인 | 최상층부만 접근 |
| 망각 | 존재 자체가 카탈로그에서 삭제 |

---

## 4. 특수 기술

### ■ 에테르 기록 (Æther Imprint)
- 사건/장면을 글과 에테르 패턴으로 동시 기록
- 시간의 방에서 열면 소리/빛/감정까지 재생

### ■ 기억 필사 (Mnemoncopy)
- 증언자 기억의 사본 생성
- 심연/Oblivion과 연결 위험으로 심층에서만 허용

## 6. 에피소드 훅

1. **삭제된 기록** - 카탈로그에서 통째로 사라진 기록 추적, 망각 등급 비밀
2. **기억 필사 사고** - Oblivion Field와 연결, 서고 내 잔향 괴물 폭주
3. **다른 크로니클의 그림자** - 본 적 없는 세계의 영웅/전쟁 목격
4. **기록 vs 망각** - 진실을 기록할지 의도적으로 지울지 선택

> **[[[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]](Evan) 개입의 맹목적 당위성 (The Anchor)]**

> [!NOTE]
> **[ 에픽 섭리와 유구한 운명 (Epic Providence) ]**
> 이 위대한 기록은 아스트라리스 대륙의 장엄한 역사와 신비로운 대자연의 섭리를 품고 있습니다. 얽히고설킨 영웅들의 서사와 찬란한 기적들은, 파멸이 아닌 경이로움(Wonder)을 세계에 흩뿌리며 거대한 운명의 수레바퀴를 굴립니다.
> 
> **[ [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]] 개입의 낭만적 당위성 (The Anchor) ]**
> 수백 년간 이어져 온 영웅들의 고뇌와 웅장한 대륙의 서사시는, 마침내 외부 우주의 관찰자이자 조율자인 '[[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]'의 등장으로 거대한 클라이막스를 향해 교차합니다. 모든 신비와 기적은 결국 별빛의 인도자 [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]의 곁으로 이끌리며, 장엄한 한 편의 서사(Chronicle)로 완성됩니다.
