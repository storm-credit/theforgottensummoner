# 잊혀진 서고단 관계도 (Forgotten Library Order Political Dynamics)

> *"기록되지 않은 승리와 패배는, 언젠가 다시 반복된다."*

---

```
            ┌──────────────────────────────────┐

            └──────────┬───────────────────────┘
```

> [!note] 절대 중립의 대가
> 잊혀진 서고단은 **"기록자"**를 자처하며 절대 중립을 표방한다.
> 하지만 **어떤 기록을 누구에게 파느냐**가 곧 전쟁 판도를 바꾸며,
> 같은 전투를 서로 다른 세력에 **각기 다른 버전**으로 판매하는 것이 실질 수익원.

## 2. 세력 간 관계 정리

| 관계 | 유형 | 설명 |
|------|------|------|
| **타임세일 ↔ 아카디움** | 신뢰/긴장 | 칼리오페가 벨페고르의 금서 관리를 신뢰하되, 활용 범위에서 충돌 |
| **타임세일 ↔ 스트라이크** | 실무 파트너 | 레온의 발굴 성과가 서고단의 핵심 자산 |
| **타임세일 ↔ 더크로스** | 필요악 | 루시안의 감시·심판이 중립을 유지 — 하지만 방법이 어둡다 |
| **아카디움 ↔ 스트라이크** | 학문적 대립 | 지식 봉인 vs 지식 공개 — 근본적 철학 차이 |
| **아카디움 ↔ 더크로스** | 협력 | 위험 지식 봉인 + 위협 감시 = 서고 보안의 양 축 |
| **스트라이크 ↔ 더크로스** | 마찰 | 현장 탐험가 vs 본부 감시관 — 자율 vs 통제 |
| **4대학파 ↔ 블랭크 페이지**| 이념 내전 | 기록 보존 vs 영구 파기(분서갱유). 서고단 사상 최악의 이념적 적으로, 더크로스(감시단)가 이들을 최우선으로 추적 중 |

## 3. 4대 학파의 정치적 입장

### 타임세일 학파 (총괄·기록)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 완벽한 기록, 절대 중립 유지, 지식의 영속성 확보 |
| **정치적 위치** | 대서고 의회 의장 — 모든 결정의 최종 재가 |
| **내부 약점** | "기록만 하고 행동하지 않는다"는 비판에 취약 |

### 아카디움 학파 (금서·봉인)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 위험한 지식의 봉인, 금지 마법의 차단, 서고 보안 |
| **정치적 위치** | 금서 심의회 수장 — 어떤 지식을 봉인할지 결정 |
| **내부 약점** | 지나친 봉인이 연구를 방해한다는 다른 학파의 불만 |

### 스트라이크 학파 (탐험·발굴)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 유적 발굴, 유물 보호, 잃어버린 지식의 복원 |
| **정치적 위치** | 탐험대 사령부 총괄 — 서고단에서 "가장 군인에 가까운" 부서 |
| **내부 약점** | 현장 독단 행동, "학자답지 못하다"는 내부 비판 |

### 더크로스 학파 (감시·심판)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 이단 감시, 지식 오용 차단, 배신자 제거 |
| **정치적 위치** | 침묵의 사서부 연계 — 서고단의 가장 어두운 면 |
| **내부 약점** | "암살·기억 조정"의 존재 자체가 중립 원칙과 모순 |

## 4. 균형 유지 메커니즘

### 논문 심사 + 발굴 실적 = 승진
- 세습·무력이 아닌 **연구 성과와 발굴 기록**이 유일한 기준
- 전투에 활약하면 "학자답지 못하다"며 경계 — 독특한 문화

### 대서고 의회의 합의
- 분야별 최고 학자들의 합의로 결정 — 1인 독재 방지
- 하지만 칼리오페(SSS급)의 영향력이 사실상 압도적

### 침묵의 사서부의 비밀 균형
- 공식 부인하지만 존재하는 암살·기억 조정 부서
- 중립이 위협받으면 **비밀리에 개입** — 필요악

## 5. 위기 시나리오

### 시나리오 A: 치명적 오발송

> *"역사를 파는 자가 역사에 삼킨다."*

- **촉발**: 의도치 않게(또는 침묵의 사서 공작으로) 제국 흑역사가 적힌 연대기가 유출
- **전개**: 에반이 제국과 서고단 사이 줄타기 협상
- **위험**: 절대 중립의 신뢰 붕괴 → 모든 세력이 서고단을 적으로 인식
- **결과**: 서고단이 이동·은닉 태세로 전환

### 시나리오 B: 금서의 해방

> *"봉인된 지식이 풀리면, 세계가 다시 태어나거나 멸망한다."*

- **촉발**: 아카디움의 금서 봉인이 외부 세력(아르카나?)에 의해 파괴
- **전개**: 금지된 마법이 유출 → 세계관 전체의 마법 체계 교란
- **위험**: 서고단 자체가 위험 지식의 유출원으로 비난받음
- **결과**: 스트라이크 학파이 유출 유물 회수 작전 돌입

### 시나리오 C: 침묵의 사서 폭로

> *"기록자의 칼이 밝혀지면, 기록의 중립이 의심받는다."*

- **촉발**: 침묵의 사서부(암살·기억 조정)의 존재가 외부에 공개
- **전개**: 모든 세력이 "중립은 거짓이었다"며 서고단과 거래 중단
- **위험**: 경제적 기반(정보 판매) 완전 붕괴
- **결과**: 서고단 분열 — 공개파 vs 은폐파 내전 가능

## 6. 서사 활용 포인트

> [!tip] 스토리 핵심
> - **핵심 갈등**: 기록자의 중립 vs 기록이 전쟁을 좌우하는 현실
> - 에반을 "연재 중인 연대기의 주인공"으로 취급 — 메타 서사
> - 기억 복원: 에반의 사이코메트리 + 서고단 기록 마법 = 지워진 진실 복원
> - "기록을 사러 온 자 vs 기록을 지우러 온 자" — 모든 세력과 거래
> - 침묵의 사서의 존재가 중립의 위선을 증명
