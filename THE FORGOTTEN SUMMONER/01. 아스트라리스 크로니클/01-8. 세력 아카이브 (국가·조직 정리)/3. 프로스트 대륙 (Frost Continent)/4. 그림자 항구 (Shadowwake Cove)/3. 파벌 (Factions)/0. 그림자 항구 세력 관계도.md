# 그림자 항구 세력 관계도 (Shadowwake Cove Power Dynamics)

> *"보스는 없다. 하지만 세 사람이 동시에 끄덕여야 일이 돌아간다."*

## 1. 4대 파벌 구조

```
            ┌──────────────────────────────────┐
            │  녹턴 파벌 (무력 · 함대 · 약탈)     │
            │ 카이우스 + 로건(돌격) + 바르칸(갑판)  │
            │ + 잭슨(약탈)                       │
            └──────────┬───────────────────────┘
                       │ 항구 의장회
          ┌────────────┼────────────┐
          │            │            │
   ┌──────┴─────┐ ┌────┴───────┐  │ (심해/비인가)
   │르블랑      │ │베인        │  │ 블랙워터 인양단
   │(재무·정보) │ │(특수·암살)   │  │ (유물·심해공작)
   │이자벨라   │ │모르굴      │  │ 데이비온(이동)
   │+세라     │ │+가레스     │  │ 
   │+로잘리아  │ │+이즈라엘   │  │

```

> [!warning] 삼두마차
> 카이우스(무력) + 이자벨라(돈) + 세라·로잘리아(정보) — 셋 중 하나라도 빠지면 항구 붕괴.
> **"보스는 없다"고 말하지만, 이 세 극점이 동시에 끄덕여야 일이 돌아간다.**

## 3. 파벌 간 관계 정리

| 관계 | 유형 | 설명 |
|------|------|------|
| **녹턴 ↔ 르블랑** | 공생 의존 | 함대가 벌고, 재무가 관리 — 돈 없이 배 못 움직임 |
| **녹턴 ↔ 베인** | 암묵적 협력 | 전투 후 뒷처리(암살·독)를 베인이 담당 |
| **르블랑 ↔ 베인** | 긴장 | 정보상(르블랑)이 너무 많이 알면 [[01-20. 직업 & 클래스 (Classes & Jobs)/직업 도감/12. 암살자 (Assassin)]](베인)가 위험 |
| **의장회 ↔ 블랙워터**| 통제 불가 | 데이비온의 심해 인양단은 3대 의장회의 명령도 씹고 독자 행동, 하지만 유물 공급 때문에 못 건드림 |

## 4. 각 파벌의 정치적 입장

### 녹턴 파벌 (무력·함대)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 해상 지배, 약탈 수익, 함대 확장 |
| **정치적 위치** | 무력 최강 — "검은 안개 함대"가 항구의 방패 |
| **내부 약점** | 카이우스의 과거가 밝혀지면 의장 자격 위기 |

### 르블랑 파벌 (재무·정보)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 경제 독점, 정보 장악, 밀수 네트워크 유지 |
| **정치적 위치** | "돈줄+정보" 이중 장악 — 실질적 최강 |
| **내부 약점** | 외부 세력(솔라리안)과의 이중 거래 의혹 |

### 베인 파벌 (특수·암흑)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 더러운 일 독점, 심연 물자 확보, 생체 실험 |
| **정치적 위치** | 두려움 기반 — "건드리면 죽는다" |
| **내부 약점** | 모르굴의 독 연구가 항구 자체를 위협할 수준 |

## 5. 균형 유지 메커니즘

### 항구 3대 규칙
1. **항구 안에서는 싸움 금지** — 위반 시 공동 축출
2. **거래는 거래로만** — 개인 원한으로 거래 방해 금지
3. **정보는 팔아도 배신은 안 된다** — 역이중첩보 시 처형

### 서열 = 배 + 돈 + 평판
- 공식 작위 없음 — "배가 크면 높고, 정보가 많으면 무섭고, 돈이 많으면 건드리지 않는다"
- 항구 규칙 위반 시 "시체로 강등"

---

## 6. 위기 시나리오

### 시나리오 A: 이중 첩자 발각

> *"정보를 파는 자가 적에게도 팔았다."*

- **촉발**: 핵심 인물 중 하나가 솔라리안 제국 스파이로 밝혀짐
- **전개**: 얼음송곳 요새 위치 노출 → 제국 해군 공격
- **위험**: 항구 전체 파괴 가능 — 12인 전원 도주 or 항전

### 시나리오 B: 흑색 깃발의 정체

> *"오벨리스크에서 온 검은 배가, 항구를 집어삼킨다."*

- **촉발**: 흑색 깃발 선박이 심연 군단의 전위 함대로 판명
- **전개**: 데이비온(잠수부대)이 이미 심연과 거래 중 → 내부 분열
- **위험**: 그림자 항구가 심연의 전진 기지로 전락

### 시나리오 C: 카이우스의 비밀

> *"검은 안개 뒤에 숨겨진 과거가, 안개보다 더 짙다."*

- **촉발**: 카이우스가 과거 솔라리안 해군 제독이었다는 사실 폭로
- **전개**: 항구 내 신뢰 붕괴 → 의장 교체 투쟁
- **위험**: 새 의장이 항구 규칙을 바꾸면 무법 지대화

## 7. 서사 활용 포인트

> [!tip] 스토리 핵심
> - **핵심 갈등**: 자유 vs 생존 — "보스가 없는 곳"이 오히려 가장 위험
> - [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]에게 "공식 루트로 갈 수 없는 곳"에 가는 유일한 방법
> - 난파선 묘역에서 사이코메트리 → 모든 대륙의 잔향을 한 곳에서 감지
> - 오벨리스크 연결 루트 → 심연 탐사의 시작점
> - 정보상 거래 → "원하는 소문은 다 있지만, 대가도 그만큼"
