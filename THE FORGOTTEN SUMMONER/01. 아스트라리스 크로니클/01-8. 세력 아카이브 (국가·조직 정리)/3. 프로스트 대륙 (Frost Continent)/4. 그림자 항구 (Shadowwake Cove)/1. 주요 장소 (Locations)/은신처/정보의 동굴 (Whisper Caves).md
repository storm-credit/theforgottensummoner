# 정보의 동굴 (Whisper Caves)

> *"여기서 팔리는 건 물건이 아니다. 누군가가 감추고 싶어하는 것이다."*

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 그림자 항구 뒤편, 검빙 절벽을 파고든 동굴 네트워크 |
| **역할** | 정보상들의 거래 장소, 각 대륙 정보 수집/판매 허브 |
| **특징** | 각 동굴이 특정 대륙/세력 정보를 전문 취급 |
| **별칭** | "속삭임의 미로", "비밀의 시장", "귀의 왕국" |

---

## 구조 & 배치

### 동굴 네트워크
- 자연 동굴을 인공적으로 확장
- 서로 연결된 12개 이상의 동굴
- 외부인은 길을 잃기 쉬움 (의도적 설계)

### 주요 동굴
| 동굴 이름 | 전문 분야 |
|------|------|
| **성국의 방** | 성국 정치, 교단 내부 정보 |
| **왕국의 방** | 왕국연합, 자유도시 정보 |
| **제국의 방** | 솔라리안 군사, 외교 정보 |
| **협회의 방** | 마법협회, 아카데미 연구 정보 |
| **프로스트의 방** | 드워프, 부족, 신전 정보 |
| **심연의 방** | 오벨리스크, 균열, Flux 관련 |
| **상업의 방** | 교역로, 물가, 상단 정보 |
| **인물의 방** | 특정 인물 추적, 신원 조회 |

### 중앙 광장 (Whisper Square)
- 동굴들이 만나는 넓은 공간
- 정보상들 간 정보 교환
- 대형 거래 협상

---

## 운영 방식

### 정보상의 규칙
1. **정확성 보장** – 거짓 정보 판매 시 자격 박탈
2. **출처 비공개** – 정보원 노출 금지
3. **역거래 금지** – 구매자를 역으로 팔면 처형
4. **가격은 협상** – 정가 없음, 가치는 상황에 따라

### 결제 방식
| 유형 | 비율 |
|------|------|
| **금화** | 30% |
| **물물교환 (다른 정보)** | 40% |
| **서비스 (향후 협력)** | 20% |
| **빚** | 10% |

### 정보 등급
| 등급 | 가격대 | 예시 |
|------|------|------|
| **일반** | 10-50 Gild | 공개된 소문, 일반 뉴스 |
| **제한** | 100-500 Gild | 내부 정보, 비공식 소식 |
| **기밀** | 1,000+ Gild | 국가 기밀, 개인 비밀 |
| **금단** | 협상 | 알려지면 안 되는 것들 |

## 주요 정보상

### 유명 정보상들
| 이름 (가명) | 전문 분야 | 특징 |
|------|------|------|
| **회색 귀** | 성국 정보 | 전직 교단 관계자 추정 |
| **그림자 혀** | 솔라리안 정보 | 제국 탈주자 네트워크 운영 |
| **얼음 눈** | 프로스트 정보 | 드워프 출신, 신뢰도 높음 |
| **잉크손** | 마법협회 정보 | 문서 위조도 가능 |
| **유령** | 인물 추적 | 누구든 찾아줌, 비쌈 |
| **무명** | 심연 정보 | 정체 불명, 오벨리스크 연결 의혹 |

### 정보상 동업
- 서로 경쟁하지만 필요시 협력
- 큰 정보는 여러 정보상이 조각씩 보유
- "완전한 그림을 원하면 여러 명에게 사라"

## 소환 & [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]] 관련

### 잔향 특성
- 수많은 **비밀, 거짓말, 협상**의 에테르 축적
- 각 대륙 정보에 담긴 원격지 잔향
- "이 동굴은 세계의 귀"

### [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]의 사이코메트리
- 동굴 벽에서: 수년간 속삭여진 비밀의 희미한 울림
- 정보상의 물건에서: 정보 수집 과정, 위험했던 순간
- 문서에서: 작성자의 의도, 숨기려 했던 것

### [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]에게 유용한 이유

- 하지만 정보상들에게 **경쟁자**로 인식될 위험
- "우리 장사를 망치는 자"로 찍힐 수 있음

### [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]] 서사 포인트

2. **경쟁자 제거** – 다른 정보상이 [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]을 배제하려 함
