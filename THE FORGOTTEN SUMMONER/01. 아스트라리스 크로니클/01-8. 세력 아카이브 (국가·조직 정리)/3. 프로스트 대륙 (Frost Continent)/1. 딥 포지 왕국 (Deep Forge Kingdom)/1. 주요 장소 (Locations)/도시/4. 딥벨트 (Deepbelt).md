# 4. 딥벨트 (Deepbelt)

> *"멈추지 마라. 흐름이 멈추면 왕국이 멈춘다."*
> — 운송 길드 슬로건

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 딥 포지 왕국 교통의 요충지, 주요 광산과 도시를 잇는 교차점 |
| **역할** | **물류 및 운송 허브 (Logistics Hub)**. 자원과 상품의 집산지 |
| **특징** | 도시 전체가 거대한 컨베이어 벨트와 화물 열차 레일로 뒤덮여 있음 |
| **별명** | "강철의 동맥", "잠들지 않는 역" |
| **이미지** | 24시간 돌아가는 기계 소음. 머리 위로는 화물 곤돌라가 지나가고, 발밑에는 고속 열차가 달리는 입체적인 교통 도시 |

### 핵심 구역
- **중앙 터미널 (Grand Terminal):** 왕국 전역으로 뻗어나가는 열차의 시발점.
- **화물 분류소 (Sorrting Zone):** 자동화 기계들이 광석과 물품을 분류하는 거대한 공장.
- **창고 거리 (Warehouse District):** 끝없이 늘어선 거대한 보관 창고들.
- **엔진 정비소 (Engine Bay):** 열차와 기계들의 심장을 고치는 곳.
- **노동자 숙소:** 3교대로 일하는 노동자들의 휴식처.

## 도시 구조 및 시스템

### 상징적 지형

| 지형 | 설명 |
|------|------|
| **무한 벨트** | 도시를 관통하는 메인 컨베이어 벨트. 길이가 수십 km에 달하며 절대 멈추지 않음. |
| **나선형 레일** | 높은 고도 차이를 극복하기 위해 나선형으로 설계된 롤러코스터 같은 철도. |
| **증기 엘리베이터** | 지하 깊은 곳과 상층부를 연결하는 고속 승강기. |

### 운송 시스템

| 시스템 | 특징 |
|------|------|
| **아이언 호스 (Iron Horse)** | 마법 공학으로 만든 고속 화물 열차. |
| **파이프라인** | 액체(용암, 물, 연료)를 운송하는 거미줄 같은 파이프망. |
| **드론 배송** | 소형 물품을 빠르게 배달하는 기계 박쥐들. |

## 정치·경제 구조

| 조직 | 역할 |
|------|------|
| **운송 노조 (Transport Union)** | 거친 노동자들의 연합. 파업하면 왕국 전체가 마비되므로 힘이 셈. |
| **레일 가드 (Rail Guard)** | 물류를 노리는 도적 떼나 몬스터로부터 열차를 지키는 무장 보안관. |
| **검수관** | 밀수품이나 불량품을 잡아내는 깐깐한 관리들. |

### 권력의 현실
- **시간은 금:** 모든 것이 시간표(Schedule)에 맞춰 돌아감. 1분의 지체도 용납하지 않는 숨 막히는 효율성.
- **지하 경제:** 공식 루트가 아닌 '비밀 터널'을 통한 밀수업이 성행함.

## 소환 · 균열 관련

### 현황
- **이동 수단:** 소환사들이 먼 거리를 이동할 때 거쳐 가는 필수 경유지.
- **몬스터 습격:** 이동 중인 화물을 노리고 지능적인 몬스터들이 터널을 습격함.

### [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]] 개입 포인트
- **[폭주 기관차]**: 브레이크가 고장 나 폭주하는 열차를, [[01-15. 인물 백과 (Character Archive)/0. 주인공 일행 (Main Characters)/에반 무영 라크라시스 (Evan)]]이 소환수(바람/염동력)를 이용해 멈춰 세움 (스파이더맨 2).
- **[터널 뚫기]**: 붕괴 사고로 막힌 터널을, 땅파기 전문 소환수(두더지?)를 이용해 뚫고 생존자를 구출.
- **[특급 배송]**: 전쟁 물자나 백신 등 긴급한 화물을, 몬스터의 방해를 뚫고 제시간에 배달하는 미션.
