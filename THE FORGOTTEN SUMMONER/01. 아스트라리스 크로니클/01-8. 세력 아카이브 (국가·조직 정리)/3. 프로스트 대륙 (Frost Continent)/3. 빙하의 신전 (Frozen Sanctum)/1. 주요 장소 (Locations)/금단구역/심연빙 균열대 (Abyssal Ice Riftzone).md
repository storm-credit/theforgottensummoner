# 심연빙 균열대 (Abyssal Ice Riftzone)

> *"이 얼음은 차갑지 않다. 텅 빈다. 마치 세상에 구멍이 난 것처럼."*

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 프로스트 대륙 동북부, 프라임 성소에서 동쪽 3일 거리 |
| **분류** | **금단 구역 (Forbidden Zone)** – 최고위 승인 외 접근 전면 금지 |
| **역할** | 심연성 에테르 오염 격리 구역, 연구 금지 구역 |
| **별칭** | "검은 빙하", "심연의 발자국", "얼어붙은 어둠" |

---

## 발생 & 역사

### 발견 (약 800년 전)
- 프라임 성소 확장 조사 중 발견
- 일반 빙하와 달리 **검은색 얼음** 지대
- 조사대 5명 중 3명이 일주일 내 실종

### 초기 조사 (700년 전)
- "심연이 얼어붙은 형태"라는 결론
- 연구 시도 → 연구자 전원 정신 붕괴
- 이후 **연구 영구 금지**

### 현재 상태
- 완전 봉쇄, 상시 감시
- 균열대가 **매년 약간씩 확장** 중 (연 2-3cm)
- 확장 억제 의식 정기 시행

---

## 지형 & 현상

### 물리적 특성
| 항목 | 설명 |
|------|------|
| **면적** | 약 5km² |
| **빙하 색상** | 순수한 검은색, 빛을 흡수 |
| **온도** | 측정 불가 (온도계가 작동 안 함) |
| **경도** | 일반 얼음보다 10배 단단함 |

### 심연 오염 현상
| 거리 | 현상 |
|------|------|
| **1km** | 불안감, 시선 느낌 |
| **500m** | 환청, 속삭임 |
| **100m** | 환각, 그림자 형체 목격 |
| **50m** | 신체 증상 (코피, 구토) |
| **접촉** | **즉사 또는 동화** |

### "살아있는 빙하"
- 균열대가 **의지를 가진 것처럼** 행동
- 가까이 오는 자를 "당기는" 느낌
- 때때로 얼음 표면에 **얼굴 형태** 나타남

## 봉쇄 시스템

### 물리적 봉쇄
| 층 | 설명 |
|------|------|
| **외곽 표식** | 1km 거리, 경고 표식 |
| **감시 초소** | 500m 거리, 교대 감시 |
| **봉인 결계** | 100m 거리, 마법 장벽 |
| **최종선** | 50m, 물리적 장벽 + 골렘 경비 |

### 봉인 의식
- 분기 1회, [[01-20. 직업 & 클래스 (Classes & Jobs)/직업 도감/08. 사제 (Cleric)]] 12인이 균열대 주변에서 동시 의식
- "심연을 재우는 기도"
- 의식 실패 시 확장 속도 급증 예상

### 긴급 대응
- 확장이 급격히 커지면: 광역 봉인 발동
- 최악의 경우: **균열대 위에 인공 빙하 붕괴** 시켜 매몰

## 내부 추정 구조

### 들어간 자들의 증언 (생환자 2명)
> "걸을수록 어두워졌다. 그리고 어둠 속에서 무언가가... 기다리고 있었다."

| 구역 | 추정 특성 |
|------|------|
| **외곽** | 일반 빙하와 검은 빙하 경계 |
| **중층** | 빛이 완전히 사라짐, 방향 상실 |
| **핵심** | 알 수 없음 (생환자 도달 못함) |

### 핵심부 가설
1. **오벨리스크 연결점**: 심연 대륙과 직접 연결
2. **봉인된 존재**: 고대 심연 생명체가 얼어있음
3. **균열의 근원**: 프로스트 모든 균열의 원천
