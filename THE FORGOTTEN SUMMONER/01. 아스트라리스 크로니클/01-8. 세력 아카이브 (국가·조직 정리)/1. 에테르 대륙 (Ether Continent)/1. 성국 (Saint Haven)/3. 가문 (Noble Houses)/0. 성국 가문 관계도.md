## 2. 가문 간 관계 정리

| 관계 | 유형 | 설명 |
|------|------|------|
| **왕가 ↔ 교황가** | 대립 | 세속 왕권 vs 종교 권력의 만성적 줄다리기 |
| **왕가 ↔ 발렌티안** | 협력 | 성왕이 군 통수권 보유, 알렉산드로가 검의 역할 |
| **교황가 ↔ 드루건** | 협력 | 교황이 이단심문소를 정적 제거에 활용 |
| **교황가 ↔ 성녀** | 통제/갈등 | 교황이 성녀를 도구로 이용, 성녀는 내면으로 저항 |
| **발렌티안 ↔ 드루건** | 긴장 | 이단 처리 방식 차이 (군사적 원칙 vs 즉각 숙청) |
| **드루건 ↔ 몬탈티** | 대립 | 숙청 강화 vs 경제적 안정 — 정면 충돌 |
| **몬탈티 ↔ 왕가** | 미묘 | 경제적 지원 제공하나, 왕가의 개혁 속도에 부담 |
| **성국 본대 ↔ 푸게토리움** | 제거 대상 | 드루건 가문의 고문에서 살아남은 탈주자들. 성국의 '정의'가 만들어낸 최악의 이단 괴물들 |

## 3. 5대 세력의 정치적 입장

### 산크투스 왕가

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 왕권 수호, 교황청 팽창 저지, 점진적 개혁 |
| **외교 노선** | 왕국연합과 군사 동맹 유지, 자유도시연합과 무역 관계 |
| **내부 약점** | 왕세자의 불확실성. 아타나시우스가 왕위를 거부하면 교황파가 개입 |

### 아이오란 교황가

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 신정 정치 강화, 왕권 약화, 이단 숙청으로 공포 정치 |
| **수단** | 드루건 가문 + 카시엘(암살) + 실라스(고문) |
| **내부 약점** | 성녀의 인기 상승 시 교황의 정당성 위협 |

### 발렌티안 가문

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 군사적 독립성 유지, 국방 우선 |
| **정치적 위치** | 왕과 교황 사이 중립 — 이 중립이 균형의 열쇠 |
| **내부 약점** | 알렉산드로가 한쪽에 기울면 균형 붕괴 |

### 드루건 가문

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 이단 완전 제거, 사법 권한 독점 |
| **정치적 위치** | 교황의 사법 파트너이나, 교황 직속 비밀 라인에 불만 |
| **내부 약점** | 교황이 발타자르를 우회하면 가문의 존재 이유가 약화 |

### 몬탈티 가문

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 경제 안정, 무역 확장, 세속적 실리 |
| **정치적 위치** | 모든 세력에 군자금·재정을 공급하는 간접적 조절자 |
| **내부 약점** | 종교적 가치를 경시한다는 비난에 취약 |

## 4. 균형 유지 메커니즘

> *"하나의 힘이 독점되지 않도록 — 그것이 성국이 유지되는 이유다."*

### 교황청의 중재
- 성녀 영향력 증가 → 드루건 가문과 협력하여 견제
- 성기사단 독립 움직임 → 사제단을 내부에 배치하여 감시
- 숙청 과잉 → 몬탈티 가문의 경제적 타격 논리로 조절

### 발렌티안 가문의 무력 균형
- 정화단 과도한 숙청 → 성기사단이 "신의 정의"를 이유로 개입
- 교황청 과도한 통제 → 명목상 충성하되 내부적으로 압박 조정
- 몬탈티 가문의 과도한 세속화 → 군사력으로 간접 압박

### 몬탈티 가문의 경제적 지렛대
- 정화단 숙청 확대 → 경제적 타격을 이유로 교황청에 개입
- 성기사단 과도한 전쟁 → 군자금 조달 제한
- 성녀의 급진 개혁 → 경제적 안정성을 이유로 속도 조절

## 5. 위기 시나리오

### 시나리오 A: 성녀 세력의 독립

> *"성녀는 신의 뜻을 따를 뿐이다. 교황의 뜻이 아니라."*

- **촉발**: 엘레오노르가 교황의 통제에서 벗어나 독자적 개혁을 시도
- **전개**: 성녀 지지 세력(일부 기사단 + 신도들)이 결집
- **위험**: 교황파 vs 성녀파로 분열 → 내전 가능성
- **외부 개입**: 왕국연합·자유도시연합이 성녀를 지지하며 개입 가능

### 시나리오 B: 성기사단 vs 정화단 충돌

> *"전사의 검과 심문관의 불꽃이 서로를 향하면, 성국은 파멸한다."*

- **촉발**: 발타자르의 무차별 숙청에 알렉산드로가 군사적으로 저항
- **전개**: 교황이 카시엘·실라스를 동원하여 성기사단 내부 와해 시도
- **위험**: 성국 군사력의 분열 → 외부 세력의 침공 기회
- **외부 개입**: 왕국연합이 성기사단을 자국 동맹으로 포섭 시도

### 시나리오 C: 외교 단절

> *"신의 이름으로 싸우는 자들에게, 신을 믿지 않는 자들은 적일 뿐이다."*

- **촉발**: 성국이 왕국연합에 강경한 종교 정책 강요
- **전개**: 자유도시연합이 무역 단절 → 몬탈티 가문 경제 위기
- **위험**: 경제 붕괴가 군사력 약화로 이어지는 연쇄 효과
- **외부 개입**: 외부 세력이 성국 내부 갈등(성녀+성기사단)과 결탁

## 6. 서사 활용 포인트

> [!tip] 스토리 핵심
> - 성녀 엘레오노르가 독자적 개혁을 시도하며 교황청과 갈등
> - 성기사단과 정화단이 대립하며 일부 기사단이 반 교황 세력으로 이동
> - 왕국연합과 자유도시연합이 성국의 내부 갈등을 이용하여 정치적 개입
> - 몬탈티 가문이 경제적 위기 속에서 교황청에 반기를 들 가능성
> - **핵심 갈등**: 빛(질서/신앙) vs 그림자(야망/타락) — 성국이라는 '빛의 나라'가 내부에서 무너지는 아이러니
