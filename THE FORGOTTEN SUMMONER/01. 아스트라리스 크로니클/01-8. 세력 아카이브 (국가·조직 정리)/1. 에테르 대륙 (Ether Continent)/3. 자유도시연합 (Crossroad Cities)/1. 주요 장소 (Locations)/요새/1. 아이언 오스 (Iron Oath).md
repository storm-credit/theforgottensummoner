# 아이언 오스 (Iron Oath)

> *"이곳의 돌 하나하나가 우리가 흘린 피와 땀의 결정체다."*
> — 요새 정초석에 새겨진 문구

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 북부, 왕국연합 국경과 맞닿은 험준한 산맥 |
| **역할** | 북부(특히 마법 제국이나 몬스터)의 위협을 막는 **공동 방어선** |
| **별명** | 강철 맹세, 북부의 방패, 3국의 눈 (왕국, 자유도시, 성국 감시단 상주) |
| **이미지** | 깎아지른 절벽 사이에 박혀 있는 회색빛 거석 요새. 만년설이 덮여 있음. |

### 핵심 시설
- **지휘 통제실 (War Room)** - 3세력 연합 지휘소
- **맹세의 탑** - 최초의 평화 조약이 체결된 장소
- **대공 방어망** - 비공정 공격을 막기 위한 마도 포대
- **지하 대피소** - 핵전쟁급 마법 공격도 견디는 벙커

## 요새 구조

### 상징적 지형

| 지형 | 설명 |
|------|------|
| **철의 관문** | 요새 정면을 막고 있는 30미터 높이의 강철 성문. |
| **얼어붙은 해자** | 요새 앞을 흐르는 강이 일 년 내내 얼어 있어 자연 해자가 됨. |
| **독수리 둥지** | 요새 최상층의 감시탑. 대륙 북부 전역이 보임. |

### 주요 구역

| 지구 | 특징 |
|------|------|
| **제1 방벽** | 최전방 방어선. 물리적 공격을 막아내는 두꺼운 성벽. |
| **제2 방벽** | 마법 방어선. **[[4. 아크브릿지]]**의 마법사들이 결계를 유지함. |
| **주둔지** | 병사들의 생활 공간. 3세력 병사들이 구역을 나눠 씀. |
| **병참 기지** | **[[6. 크로스로드]]**에서 올라온 보급품이 쌓이는 거대한 창고. |

## 군사·행정 구조

| 기관 | 역할 |
|------|------|
| **연합 사령부** | 왕국연합 장군, 자유도시 용병단장, 성국 성기사단장이 공동 지휘. |
| **마도 공병대** | **[[10. 에드윈 카르다스]]**가 보낸 폭발물 전문가들이 방어선 구축. |
| **보급 장교** | 물자 배급 담당. 횡령을 막기 위해 3세력이 교차 감시함. |

### 권력의 현실
> "적보다 아군을 더 경계해야 하는 곳."

- 겉으로는 '연합'이지만, 실제로는 서로가 서로를 감시하는 긴장 상태.
- 왕국연합은 명분을, 자유도시는 실리(돈)를, 성국은 위상(명예)을 챙기려 함.
- 비상시 지휘권을 누가 가질 것인가가 항상 논란거리.

## 주둔 병력

| 부대 | 특징 |
|------|------|
| **왕국 기사단** | 정규 훈련을 받은 중장보병 및 기사. 규율이 엄격함. |
| **자유 용병대** | **[[2. 머시너리 게이트]]**에서 파견된 베테랑들. 실전 경험이 풍부함. |
| **수호 사제단** | 성국 파견. 부상자 치료 및 언데드 퇴치 담당. |

### 특수 상황
- **보급 전쟁:** 겨울이 되면 폭설로 보급로가 끊겨 식량 쟁탈전이 벌어지기도 함.
- **몬스터 웨이브:** 북부 설원에서 내려오는 예티나 아이스 트롤의 대규모 침공.

| 시간 | 모습 |
|------|------|
| **점호** | 각기 다른 군복을 입은 병사들이 섞여서 점호를 받음. |

> "춥고, 배고프고, 서로를 믿지 못하지만, 그래도 등을 맞대고 싸워야 함."

## 소환 · 균열 관련

### 공식 입장
- **"전시 상황에서는 모든 수단을 동원."**
- 소환수는 '전략 병기'로 취급되어 귀한 대접을 받음.

### 에반 개입 포인트
- **지휘관 중재:** 3세력 지휘관들의 다툼으로 방어선이 뚫릴 위기. 에반이 카리스마로 통합 지휘.
- **내부의 적:** 요새 안에 숨어든 마법 제국의 스파이가 결계석을 파괴하려 함.
- **설원 정찰:** 폭설 속에서 실종된 정찰대를 찾기 위해 감각이 예민한 소환수 활용.
