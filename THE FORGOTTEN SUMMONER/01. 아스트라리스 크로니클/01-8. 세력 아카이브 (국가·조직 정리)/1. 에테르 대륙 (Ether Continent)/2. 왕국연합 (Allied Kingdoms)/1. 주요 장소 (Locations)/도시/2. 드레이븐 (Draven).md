# 드레이븐 (Draven)

> *"검보다 혀가 더 날카로운 곳."*

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 왕국연합 서부 국경, 정령연합·자유도시 삼거리 |
| **역할** | 외교·상업 중심지, 주요 조약 체결지 |
| **별명** | 외부: "혀의 도시" / 내부: "거짓말의 광장" |
| **이미지** | 삼색 깃발이 나부끼는 원형 광장, 다국어가 뒤섞인 거리 |

### 핵심 시설
- 외교관 회관 (Diplomat's Hall)
- 국제 시장 (International Market)
- 통역사 길드
- 삼국 조약탑 (Three Nations Tower)
- 밀사관 (Secret Envoy House)

## 정치·행정 구조

| 기관 | 역할 |
|------|------|
| **삼국 조율 위원회** | 왕국연합+정령연합+자유도시 대표 참여, 도시 운영 조율 |
| **도시 총독** | 왕국연합 임명, 치안·세금 총괄 |
| **외교관 평의회** | 각국 대사 참여, 분쟁 조정 |

### 권력의 현실
> "총독은 이름뿐, 진짜 권력은 밀사들의 귓속말에서 결정된다."

- 공식적으로는 왕국연합 영토
- 실제로는 **세 세력 모두 영향력 행사**
- 정보와 밀약이 금화보다 가치 있는 도시

## 군사 & 치안

| 조직 | 역할 |
|------|------|
| **드레이븐 수비대** | 왕국연합 파견, 도시 치안 |
| **외교관 호위대** | 각국 대사관 자체 경비 |
| **통역사 길드 경비** | 길드 보호 + 정보 보안 |

### 특수 상황
- 외교관 면책권으로 범죄 처벌 복잡
- 암살 시도 빈번 → 식음료 검사 철저
- 세 세력 모두 자체 정보망 운영

## 신앙 · 일상 · 분위기

### 신앙
- 세 세력 신앙이 **모두 공존**
- 솔라리스 신전, 정령 제단, 자유도시 행운신 사당 나란히
- "신도 여기선 협상한다"는 농담

### 일상 풍경
| 시간 | 모습 |
|------|------|
| 아침 | 대사관 마차 행렬, 통역사 출근 |
| 낮 | 시장 흥정, 공식 회담, 조약 서명식 |
| 저녁 | 연회, 비공식 회동, 술자리 협상 |
| 밤 | 밀사 이동, 정보 거래, 암살 시도 |

### 도시 정서
> "모두가 미소 짓지만, 모두가 뒷조사를 하고 있다."

## 소환 · 균열 관련

### 공식 입장
- 세 세력 합의: **"외교 중립 지대에서 소환 사용 금지"**
- 균열 발생 시 공동 대응 조약 존재

### 실제 상황
| 세력 | 현실 |
|------|------|
| **왕국연합** | 밀사가 소환사 정보 수집 |
| **정령연합** | 정령과 소환의 관계 연구 관심 |
| **자유도시** | 소환사를 "특수 자산"으로 거래 시도 |

### 에반 개입 포인트
- 삼국 조약 갱신 협상 시 **균열 대응 조항** 논의 증인
- 특정 세력이 에반을 **비밀리에 접촉** 시도
- 밀사 암살 사건의 **진범 추적**에 능력 활용

| **통역사 길드장** | 모든 언어와 비밀을 아는 인물 | 정보 제공 대가로 협력 |
| **그늘의 중개인** | 밀사 세계의 실세 | 위험한 의뢰, 뒷거래 |

## 개연성 & 서사 메모

> [!note] 개연성
> 세 세력 교차점이라는 지리적 특성이 외교 도시 형성의 필연성 부여.
> 노블가르드(정치)와 달리 다국적 외교·첩보 중심.

> [!important] 서사 활용
> - 삼국 조약 갱신 시즌에 **암살 + 음모** 에피소드
> - 밀사 암살 사건 → 에반의 사이코메트리로 진범 추적
> - 세 세력 모두 에반을 **자기 편으로** 끌어들이려는 경쟁
