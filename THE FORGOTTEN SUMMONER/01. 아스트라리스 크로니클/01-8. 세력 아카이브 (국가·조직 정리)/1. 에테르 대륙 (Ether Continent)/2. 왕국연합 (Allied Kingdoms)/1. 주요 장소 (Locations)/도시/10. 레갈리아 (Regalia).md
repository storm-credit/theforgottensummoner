# 레갈리아 (Regalia)

> *"왕들의 맹약으로 서쪽을 지킨다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 중서부, 연합 중심 |
| **역할** | 왕국연합 수도, 정치·외교 중심지 |
| **특징** | 연합 왕 의회, 대사관, 조약 회의장 |
| **이미지** | 5개 왕국 깃발이 나란히 펄럭이는 대화의 도시 |

---

## 2. 지형·구조

### ■ 지리적 특성
- **중앙 위치**: 어느 왕국에서도 접근 가능한 중립 지점
- **강 교차점**: 여러 강이 만나는 교통 요충지
- **개방형 도시**: 군사 요새보다 외교/상업 중심 설계

### ■ 주요 구역
| 구역 | 특징 |
|------|------|
| **연합 의회관 (Council Hall)** | 5대 왕국 대표 회의장, 최고 의사결정 장소 |
| **대사관 구역** | 각 세력 공식 대표부 밀집 |
| **왕의 광장** | 조약 체결, 연합 선언 공식 장소 |
| **상업 구역** | 연합 최대 시장, 각 왕국 특산품 교류 |
| **귀족 저택 구역** | 연합 귀족/외교관 거주지 |

---

## 3. 정치·외교

### ■ 순환 의장제
| 특성 | 설명 |
|------|------|
| **5대 왕국** | 브룬발트, 리오나르, 카르디온 등 순환 |
| **임기** | 2년 순환 (형식적으로 동등) |
| **실제 권력** | 군사력+재정+혈통 순 발언권 |

### ■ 주요 기관
| 기관 | 역할 |
|------|------|
| **연합 왕 의회** | 최고 의결 기관 |
| **외교부** | 타 세력(성국, 자유도시 등) 외교 |
| **첩보부** | 비밀 정보 조직 |

---

## 4. 문화

### ■ 행사
| 행사 | 설명 |
|------|------|
| **연합 시합** | 각 왕국 기사 대표 토너먼트 |
| **조약 축제** | 평화 조약 기념 연례 행사 |
| **귀족 살롱** | 정치·문화·혼인 외교의 장 |

## 5. 에반 관련

### ■ 에반에게 의미
- **외교 창구**: 왕국연합 공식 접촉 가능
- **정보 수집**: 각 왕국 동향 파악
- **정치적 복잡함**: 음모와 동맹이 뒤얽힌 곳

### ■ 접점 시나리오
| 케이스 | 시나리오 |
|--------|----------|
| **외교 협상** | 왕국연합과 공식 교섭 |
| **기사 시합** | 토너먼트에 참가/관전 |
| **음모 탐지** | 정치적 암살/배신 계획 저지 |
