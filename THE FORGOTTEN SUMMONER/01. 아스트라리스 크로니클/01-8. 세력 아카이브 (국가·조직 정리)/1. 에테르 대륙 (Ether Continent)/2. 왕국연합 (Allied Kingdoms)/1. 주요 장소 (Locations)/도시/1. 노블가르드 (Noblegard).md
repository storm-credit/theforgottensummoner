# 노블가르드 (Noblegard)

> *"명예는 검 끝에서 나오고, 질서는 왕좌에서 시작된다."*

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 서부 구릉지대, 5대 왕국의 교차점 |
| **역할** | [[왕국연합 (Allied Kingdoms)]] 정치적 수도 + 귀족 의회 소재지 |
| **별명** | 외부: "왕들의 도시" / 내부: "음모의 광장" |
| **이미지** | 하얀 대리석 성벽과 붉은 지붕군, 높이 솟은 왕국 연합 깃발들이 휘날리는 고전적인 요새 도시 |

### 핵심 시설
- 귀족 의회 광장 (Noble Assembly Plaza)
- 기사 아카데미 (Knight Academy)
- 5왕국 대사관 구역
- 중앙 재판소
- 왕관 보관소

| **백색 성벽 (White Walls)** | 3중 성벽 시스템. 각 층마다 다른 왕국 기사단이 교대 경비 |

| **상인 회랑 (Merchant Corridor)** | 외곽 상업 지구. 자유도시 상단과 각 왕국 상단 경쟁 |

| **하층민 구역** | 성벽 바깥 임시 거주지. 용병, 하인, 떠돌이 모험가 |

## 정치·행정 구조

| 기관 | 역할 |
|------|------|
| **귀족 의회** | 5대 왕국 대표 + 주요 영주 참여. 연합 정책, 전쟁, 조약 의결 |
| **순번제 의장** | 3년 주기로 5왕국 왕이 교대. 외교 대표 + 회의 주재 |
| **왕국 재판소** | 영주 간 분쟁, 국제 상거래 분쟁 조정 |
| **기사단 평의회** | 각 왕국 기사단장 참여. 군사 문제 협의 |

### 권력의 현실
> "의장은 이름뿐, 진짜 권력은 회의 사이에 오가는 밀약에서 결정된다."

- 공식적으로는 평등한 5왕국 연합
- 실제로는 **가장 강한 군사력 + 가장 많은 금화**를 가진 왕국이 주도
- 성국과의 관계: 종교적으로는 거리를 두면서도 군사 동맹 유지

## 군사 & 치안

| 조직 | 역할 |
|------|------|
| **노블가르드 수비대** | 도시 치안. 5왕국에서 순환 파견 |
| **왕관 근위대** | 의회 + 의장 직접 경호 |
| **기사 아카데미 훈련생** | 비상 시 예비 병력. 실전 경험용 순찰 |
| **용병 조합** | 성벽 외곽. 비공식 일감 처리, 정보 거래 |

### 방어 체계
- 3중 백색 성벽 + 각 문마다 다른 왕국 문장
- 비상 시 모든 왕국 기사단 소집 체계
- 지하 비밀 통로 (의장 탈출용, 위치 극비)

## 신앙 · 일상 · 분위기

### 신앙
- 성국 솔라리스 신앙은 **공식적으로 존중, 실질적으로 거리 유지**
- 각 왕국 고유 수호신/조상 숭배가 더 강함
- 도시 내 신전은 있으나 성국만큼 화려하지 않음

### 일상 풍경
| 시간 | 모습 |
|------|------|
| 아침 | 기사 아카데미 훈련 소리, 귀족 마차 행렬 |
| 낮 | 의회 회의, 상인 거래, 결투장 구경꾼 |
| 저녁 | 귀족 연회, 기사 주점의 술자리, 하층민 구역 도박 |
| 밤 | 대사관 간 비밀 밀사, 용병 조합 야간 계약 |

### 도시 정서
> [!note] 분위기
> "모두가 예의 바르지만, 모두가 뭔가를 숨기고 있다."
> 겉은 명예와 기사도, 속은 영지 경쟁과 이권 다툼.

## 소환 · 균열 관련

### 공식 입장
- 성국보다 유연: **"위험하지만 완전 금지는 아님"**
- 각 왕국 마법사/연구원이 개별 연구 허용
- 단, 공개 소환은 의회 허가 필요

### 실제 상황
| 계층 | 현실 |
|------|------|
| **귀족** | 개인 마법사를 통해 비밀 소환 연구. 전쟁 무기로 관심 |
| **기사단** | "소환은 명예롭지 못하다" 공식 입장, 실제론 호기심 |
| **용병 조합** | 소환사 고용·중개. 높은 값에 비밀 유지 |

### 에반 개입 포인트
- 의회에서 **"소환술 규제안"** 논의 시 증인/전문가로 소환
- 특정 왕국이 에반을 **비밀 병기/자문**으로 영입 시도
- 기사 아카데미 학생과 **소환 vs 검술** 철학 논쟁

## 개연성 & 서사 메모

> [!note] 개연성
> 성국의 종교 수도 루멘 vs 왕국연합의 정치 수도 노블가르드
> → 두 도시 방문 시 **"신앙 vs 권력"** 대비 구조

> [!important] 서사 활용
> - 의회 회기 중 **암살 시도 + 정치 음모** 에피소드
> - 기사 아카데미 **경연대회**에 소환 영웅 참가
> - 5왕국 간 **전쟁 직전 긴장** 속 중재자 역할
