# 골드리프 (Goldleaf)

> *"밀밭이 황금빛으로 물들면, 대륙 전체가 배불리 먹는다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 중앙 평야 지대 |
| **역할** | 왕국연합 최대 농업 왕국, 곡창 지대 |
| **특징** | 끝없는 밀밭, 풍요의 축제, 평화로운 분위기 |
| **이미지** | 황금빛 들판 위에 떠 있는 붉은 지붕 마을들 |

---

## 2. 지형·구조

### ■ 지리적 특성
- **광활한 평야**: 에테르 대륙 최대 경작지
- **강 네트워크**: 풍부한 관개 시설
- **분산된 마을**: 거대 도시 대신 수백 개 마을 연결

### ■ 주요 구역
| 구역 | 특징 |
|------|------|
| **황금관 (Goldencrown)** | 왕궁이 있는 중앙 도시, 수확 축제 장소 |
| **곡창 지대** | 밀, 보리, 과일 대규모 재배 |
| **양조 마을** | 맥주, 와인 생산 전문 지역 |
| **목축 구역** | 소, 양, 말 방목 지대 |
| **강변 시장** | 농산물 집하/거래 중심지 |

---

## 3. 정치·군사

### ■ 통치 구조
| 직위 | 역할 |
|------|------|
| **곡물왕 (Harvest King)** | 세습 왕, 농업 정책 총괄 |
| **마을 장로 연합** | 각 마을 대표, 실질적 지방 자치 |
| **수확 사제단** | 풍요의 신 신앙 + 농업 기술 보존 |

### ■ 군사력
- **약함**: 상비군 최소, 전시 농민 징집
- **방어 전략**: 다른 왕국에 식량 제공 → 보호 받음
- **곡물 방패**: "우리를 공격하면 대륙이 굶는다"

---

## 4. 경제

### ■ 주요 산업
| 산업 | 설명 |
|------|------|
| **곡물 생산** | 연합 전체 식량의 60% 공급 |
| **축산** | 육류, 유제품, 가죽 |
| **양조** | 맥주, 와인 수출 |

### ■ 강점/약점
- **강점**: 모든 세력이 골드리프에 의존
- **약점**: 군사력 부재, 흉년 시 위기

---

## 5. 문화

### ■ 주요 축제
| 축제 | 시기 |
|------|------|
| **황금 수확제** | 가을, 대륙 최대 농업 축제 |
| **씨앗 축복식** | 봄, 새 시즌 시작 의식 |
| **맥주 축제** | 여름, 신규 양조 발표 |

### ■ 분위기
- 평화롭고 느긋한 생활
- 전쟁과 정치에 무관심
- "일하고, 먹고, 마시고, 자라"

---

## 6. 에반 관련

### ■ 에반에게 의미
- **휴식처**: 전투 없는 평화로운 환경
- **정보 부족**: 정치/군사 정보 얻기 어려움
- **식량 확보**: 장거리 여정 준비

### ■ 접점 시나리오
| 케이스 | 시나리오 |
|--------|----------|
| **휴식** | 전투 후 회복차 방문 |
| **흉년 위기** | 균열/몬스터로 농지 피해, 구조 |
| **곡물 독점** - 상인/세력이 곡물 장악 시도, 저지 |

---

## 7. 에피소드 훅

1. **들판의 균열** - 밀밭 한가운데 균열 발생, 수확 전 해결 필요
2. **흉년의 저주** - 갑작스런 흉작, 원인 조사 (마법/저주/몬스터)
3. **식량 전쟁** - 다른 세력이 곡창 장악 시도, 방어
4. **수확제의 암살** - 축제 중 고위 인사 암살 계획 저지
