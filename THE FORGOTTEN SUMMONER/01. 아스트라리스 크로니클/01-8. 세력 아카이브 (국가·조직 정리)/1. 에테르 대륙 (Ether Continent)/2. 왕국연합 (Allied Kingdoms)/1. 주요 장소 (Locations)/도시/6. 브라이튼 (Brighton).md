# 브라이튼 (Brighton)

> *"풍요로운 대지가 왕국의 힘이 된다."*

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 왕국연합 중앙 평원, 비옥한 강 유역 |
| **역할** | 농업·식량 생산 중심지, 대륙 곡창 |
| **별명** | 외부: "황금 밀밭" / 내부: "땀의 도시" |
| **이미지** | 끝없이 펼쳐진 황금빛 밀밭, 거대한 저장고들, 농부들의 노래 |

### 핵심 시설
- 황금 밀밭 (Golden Wheat Fields)
- 농업 조합 본부
- 중앙 저장고 구역 (Granary District)
- 수확제 광장
- 농업 마법 연구소

## 도시 구조

### 상징적 지형

| 지형 | 설명 |
|------|------|
| **황금 평원 (Golden Plains)** | 도시 주변 광활한 밀밭. 수확기엔 금빛 바다 |
| **강둑 수로 (Riverbank Canals)** | 관개 시스템. 농업 마법으로 수량 조절 |
| **저장고 언덕 (Granary Hill)** | 대규모 곡물 저장 시설 밀집 |
| **수확제 광장 (Harvest Square)** | 연례 축제 장소, 농민 집결지 |

### 주요 지구

| 지구 | 특징 |
|------|------|
| **농업 조합 지구** | 조합 본부, 가격 책정, 상단 협상 |

| **농민 거주지** | 소박한 주거, 가족 농장 |
| **상인 지구** | 곡물 거래, 대상 집결지 |

## 정치·행정 구조

| 기관 | 역할 |
|------|------|
| **농업 조합 의회** | 대농장주 + 소농 대표. 생산 할당, 가격 협상 |
| **도시 영주** | 왕국연합 귀족. 세금 징수, 치안 |
| **수확 관리관** | 저장고 관리, 비상 식량 배분 |

### 권력의 현실
> "밀 가격을 정하는 자가 진짜 권력자다."

- 대농장주 vs 소농 갈등 상존
- 흉년 시 폭동 위험
- 식량 = 정치적 무기

## 군사 & 치안

| 조직 | 역할 |
|------|------|
| **브라이튼 민병대** | 농민 출신 자경단, 수확기 경비 |
| **저장고 경비대** | 곡물 창고 전담 보안 |
| **영주 수비대** | 왕국연합 파견 정규군 |

### 특수 상황
- 수확기엔 도적·약탈자 증가
- 저장고는 **왕국연합 전략 자산**으로 보호
- 흉년 시 폭동 진압 준비

---

## 신앙 · 일상 · 분위기

### 신앙
- **대지와 수확의 신** 숭배가 주류
- 솔라리스 신앙은 부차적
- 매 수확기마다 **수확제** 대축제

### 일상 풍경
| 시간 | 모습 |
|------|------|
| 새벽 | 농부들 밭으로 출발, 닭 우는 소리 |
| 낮 | 밭일, 관개 작업, 상단 거래 |
| 저녁 | 귀가, 가족 식사, 술집 모임 |
| 수확기 | 온 도시가 축제 분위기, 노래와 춤 |

### 도시 정서
> "여기선 땀 흘린 만큼 거둔다. 음모는 도시 것들 짓이다."

## 소환 · 균열 관련

### 공식 입장
- 농민들: **"그런 복잡한 건 관심 없다"**
- 영주: 균열이 농지 피해 시에만 관심

### 에반 개입 포인트
- 균열이 **농지 오염** 시 정화 요청
- 흉년/기근 → **비밀 소환사가 해결책?** 루머
- 수확제에서 **예언/축복** 역할 제안
