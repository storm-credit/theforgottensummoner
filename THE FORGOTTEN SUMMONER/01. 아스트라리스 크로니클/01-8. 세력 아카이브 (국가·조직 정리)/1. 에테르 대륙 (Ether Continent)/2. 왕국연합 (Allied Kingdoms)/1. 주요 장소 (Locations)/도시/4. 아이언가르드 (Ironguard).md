# 아이언가르드 (Ironguard)

> *"강철은 녹슬지 않고, 철의 맹세는 깨지지 않는다."*

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 동부 산악 지대, 크림슨 대륙 접경 |
| **역할** | 왕국연합 최전방 군사 왕국 |
| **특징** | 철광산, 대장장이 전통, 정예 중기병 |
| **이미지** | 산 위의 석조 요새 도시, 끊임없는 망치질 소리와 군가 |

---

## 2. 지형·구조

### ■ 지리적 특성
- **산악 요새 도시**: 자연 방벽을 활용한 다층 성채
- **철광 벨트**: 도시 하부에 거대한 광산 네트워크
- **협곡 관문**: 크림슨 대륙으로 통하는 유일한 육로 통제

### ■ 주요 구역
| 구역 | 특징 |
|------|------|
| **왕성 (Iron Crown)** | 산 정상부, 철로 장식된 왕궁 |
| **대장간 거리** | 왕국 최고 무기/갑옷 생산지 |
| **병영 구역** | 상시 주둔 5만 대군 |
| **광산 지구** | 지하 채굴 도시, 민간인 다수 |
| **관문 요새** | 크림슨 방향 3중 성문 방어선 |

---

## 3. 정치·군사

### ■ 통치 구조
| 직위 | 역할 |
|------|------|
| **철왕 (Iron King)** | 군사·행정 최고 책임자 |
| **철의 원로원** | 군 장성·광산 귀족·대장장이 길드 대표 |
| **관문 총독** | 크림슨 접경 방어 전담 |

### ■ 군사력
- 왕국연합 **최강 육군** 보유
- **철의 기사단**: 중장갑 기병 정예
- **산악 보병**: 험지 전투 특화
- **공성 병기**: 발리스타, 투석기 대량 보유

### ■ 왕국연합 내 역할
> "아이언가르드가 무너지면 연합이 무너진다"

- 크림슨 침공 방어 최전선
- 연합 군사 회의 주도권 보유
- 다른 왕국에 무기 공급

---

## 4. 경제

### ■ 주요 산업
| 산업 | 설명 |
|------|------|
| **철광 채굴** | 대륙 최대 철광 생산 |
| **무기 제조** | 연합 전체 군수품 공급 |
| **용병 수출** | 퇴역 군인들의 용병 활동 |

### ■ 약점
- 농업 생산 부족 → 식량 수입 의존
- 광산 의존 경제 → 고갈 시 위기

---

## 5. 에반 관련

### ■ 에반에게 의미
- **크림슨 진입로**: 육로로 크림슨 가려면 통과 필수
- **군사 정보**: 전선 상황 파악 가능
- **소환에 대한 경계**: 군사 국가라 마법/소환에 보수적

### ■ 접점 시나리오
| 케이스 | 시나리오 |
|--------|----------|
| **통과 허가** | 크림슨 진입을 위한 외교 |
| **전선 투입** | 균열/몬스터 사태에 협력 요청 |
| **군사 기밀** | 철왕이 감추는 비밀 조사 |

---

## 6. 에피소드 훅

1. **관문 위기** - 크림슨에서 대규모 공격, 방어선 지원
2. **광산의 비밀** - 깊은 광산에서 고대 유적/괴물 발견
3. **철왕의 야심** - 연합 주도권 장악 시도, 내부 갈등
4. **탈영병 추적** - 중요 정보 가진 탈영병 추적/보호
