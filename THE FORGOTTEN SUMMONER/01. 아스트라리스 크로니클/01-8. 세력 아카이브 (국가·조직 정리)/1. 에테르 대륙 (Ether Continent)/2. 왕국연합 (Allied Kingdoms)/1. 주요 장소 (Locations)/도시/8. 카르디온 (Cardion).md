# 카르디온 (Cardion)

> *"금화가 흐르는 곳에 권력도 흐른다."*

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 에테르 대륙 남부, 왕국연합-자유도시 접경 |
| **소속** | [[왕국연합 (Allied Kingdoms)]] 핵심 3왕국 중 하나 |
| **별칭** | 연방시, 경제 왕국 |
| **역할** | 부유한 도시·소영지 연합. 자유도시연합과도 깊게 연결된 경제 허브 |
| **이미지** | 강변 창고, 상선 깃발, 은행과 상회가 즐비한 거리 |

## 지형 & 도시 구조

### 상업 도시
- **실버린 강** 유역의 주요 항구 도시
- 자유도시연합과 경계가 모호한 지역들 존재
- 귀족보다 상인 길드의 영향력이 강함

### 주요 구역
| 구역 | 특징 |
|------|------|
| **왕궁 (Gold Crown)** | 소박한 왕궁, 실권은 상의원에 |
| **대상회 거리** | 왕국 최대 상단 본부 밀집 |
| **강변 창고지구** | 실버린 강 물류 중심 |
| **은행가** | 금융 거래, 환전, 어음 발행 |
| **연합 접경 시장** | 자유도시 상인들과 혼합 거래 |

## 정치 & 경제

### 통치 구조
| 직위 | 역할 |
|------|------|
| **카르디온 왕** | 명목상 군주, 실권 제한적 |
| **상의원 (Merchant Council)** | 주요 상단 대표들의 의회, 실질 정책 결정 |
| **세금 감찰관** | 연합 분담금 관리 |

### 경제력
- 왕국연합 내 **최대 상업·금융 중심**
- 자유도시연합과 **삼각 교역** 핵심 축
- 전쟁 시 **군자금 제공** → 정치적 발언권

### 약점
- 군사력 약함 → 다른 왕국에 방어 의존
- 상인 이익 우선 → 연합 결속력 약화 요인

## 군사력

### 카르디온 방위군
- 규모 작음, 도시 경비 수준
- **용병 고용**으로 전력 보충
- 강변 수비대가 주력

### 연합 기여
- 군사력보다 **재정 지원**으로 역할
- 전쟁 어음, 군수품 조달
