# 정령연합 세력 관계도 (Spirit Union Political Dynamics)

> *"우리에게 왕좌는 없다. 다만, 나무가 기억하는 자가 말할 권리를 가진다."*

---

> 세습이 아닌 **정령의 선택**에 의해 지위가 결정된다.
> 만장일치 합의제로 운영되어, 합의가 안 나면 **아무것도 실행되지 않는다.**

## 2. 세력 간 관계 정리

| 관계 | 유형 | 설명 |
|------|------|------|
| **엘도라 ↔ 실버브레이드** | 군주-기사 | 정령왕의 방패. 알라리안이 실바리온의 가장 충실한 검 |
| **엘도라 ↔ 썬더루트** | 군주-현자 | 정령왕의 지혜. 드라이덴의 조언이 정책 결정의 근간 |
| **엘도라 ↔ 그린스톰** | 협력적 긴장 | 수인족의 자치권 vs 연합 통합 — 발린의 독립심이 변수 |
| **실버브레이드 ↔ 썬더루트** | 상호 보완 | 검과 지혜 — 전투와 치유의 완벽한 짝 |
| **실버브레이드 ↔ 그린스톰** | 군사적 동맹 | 엘프 전사 + 수인 돌격대 = 숲의 최강 전력 |
| **썬더루트 ↔ 그린스톰** | 문화적 마찰 | 드루이드의 절제 vs 수인족의 본능 — 「숲의 뜻」 해석 차이 |
| **정령연합 ↔ 블랙 우드** | 내부의 암 | 보호받아야 할 자연이 독과 역병으로 타락. 썬더루트(치유)가 이들을 막고 정화하려 사투 중 |

## 3. 4대 세력의 정치적 입장

### 엘도라 (정령왕)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 숲의 보존, 정령과의 계약 유지, 외부 세력 침입 방어 |
| **외교 노선** | 자유도시연합과 실리적 거래, 나머지와는 경계 |
| **내부 약점** | 합의제의 한계 — 빠른 결단이 필요할 때 치명적 느림 |

> [!note] 실바리온의 고독
> 실바리온 엘도라는 **수천 년을 살아온 장수종 엘프**로, 숲의 기억을 직접 들을 수 있다.
> 하지만 너무 오래 살아온 자의 피로가 있으며,
> "다음 세대에게 숲을 맡길 수 있을까?"라는 불안이 항상 따라다닌다.

### 실버브레이드 가문 (전사)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 숲의 물리적 방어, 침입자 격퇴, 성지 보호 |
| **정치적 위치** | 정령왕의 검이자 연합의 군사적 핵심 |
| **내부 약점** | 실라스(나이트쉐이드)의 충성이 숲이 아닌 클랜에 있을 가능성 |

### 썬더루트 가문 (드루이드)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 자연 균형 유지, 치유 의식, 고대 지식 보존 |
| **정치적 위치** | 정령왕의 지혜이자 연합의 정신적 기둥 |
| **내부 약점** | 절제와 평화주의가 과해지면, 군사적 위협에 대응 불가 |

> [!note] 드라이덴의 원칙
> 드라이덴 썬더루트는 **\"숲은 스스로를 지킨다\"**는 신념의 소유자다.
> 극단적 비폭력을 추구하여, 전쟁 시 치유와 의식에만 집중한다.
> 이 원칙이 숲의 위기 시 군사적 대응을 늦추는 원인이 되기도 한다.

### 그린스톰 가문 (수인족)

| 항목 | 내용 |
|------|------|
| **핵심 목표** | 수인족 자치권 보장, 전장에서의 존재감 확보 |
| **정치적 위치** | 연합에서 가장 전투적인 세력 — 「숲의 이빨」 |
| **내부 약점** | 수인족의 본능적 공격성이 드루이드의 절제와 충돌 |

## 4. 균형 유지 메커니즘

> *"숲은 균형으로 유지된다. 우리도 마찬가지다."*

### 엘도라의 합의 중재

- 실버브레이드가 독단적 군사 행동 시도 → **대정령회에서 만장일치 부결로 차단**
- 그린스톰이 자치권 강화 요구 → **정령왕이 직접 수인족 부족 방문하여 대화**
- 외부 세력의 침입 → **정령왕의 소집으로 전 구역 연합 방어 발동**

### 실버브레이드의 무력 균형

- 성국의 숲 개간 시도 → **정령 결계 + 전사단의 물리적 방어 이중 차단**
- 그린스톰의 무모한 돌격 제어 → **알라리안이 직접 전선 지휘하여 통합 작전**
- 내부 분열 시 → **실라스의 첩보가 균열의 원인을 조기 파악**

### 썬더루트의 정신적 균형

- 전쟁의 광기가 확산 → **치유 의식과 정령 소통으로 연합의 정신적 중심 유지**
- 정령이 분노할 때 → **드루이드가 중재하여 자연의 폭주 방지**
- 숲의 기억이 왜곡될 때 → **고대 의식으로 진실 복원**

## 5. 위기 시나리오

### 시나리오 A: 정령왕의 은퇴

> *"나무는 영원하지만, 왕은 아니다."*

- **촉발**: 실바리온이 숲의 피로를 느끼고 정령왕직 사퇴 결심
- **전개**: 알라리안(전사) vs 드라이덴(현자) vs 발린(수인족) — 후계 3파전
- **위험**: 합의제로 후계가 결정되지 않으면 연합 분열 → 각 구역 독립
- **외부 개입**: 성국이 "이교도" 분열의 틈을 노려 대규모 침공

### 시나리오 B: 나이트쉐이드의 배신

> *"숲의 그림자가 적의 그림자와 하나였다면?"*

- **촉발**: 실라스 나이트쉐이드가 **클랜 본가의 지령**에 따라 정령연합 내부 정보 유출
- **전개**: 칼레스트(마법협회), 자렉(용병단) 등 같은 클랜원에게 숲의 방어 취약점 전달
- **위험**: 정령 결계의 비밀이 노출 → 숲의 무적 방어가 무력화
- **외부 개입**: 마법협회가 정령 결계 해제 기술을 개발하여 침입

### 시나리오 C: 숲의 분노 — 정령의 직접 개입

> *"숲이 깨어나면, 누구도 막을 수 없다."*

- **촉발**: 균열·오염·개간이 임계점을 넘어 상급 정령이 직접 전쟁 선포
- **전개**: 정령연합도 통제 불능 — "우리가 말려야 하나, 함께 싸워야 하나?"
- **위험**: 자연 재해 수준의 대파괴 → 주변 국가 도시까지 숲이 삼킴
- **외부 개입**: 모든 세력이 연합하여 정령을 진정시키려 하나, 정령은 대화를 거부

## 6. 서사 활용 포인트

> [!tip] 스토리 핵심
> - **핵심 갈등**: 보존(자연) vs 개발(문명) — 누가 진짜 침략자인가?
> - 실바리온의 수천 년 피로와 후계 문제가 연합 존속의 시한폭탄
> - 수인족(그린스톰)의 본능적 전투성이 드루이드(썬더루트)의 절제와 충돌
> - **나이트쉐이드 클랜**: 실라스(정령연합) · 칼레스트(마법협회) · 자렉(용병단) → 대륙 전역 비밀 네트워크
> - 정령이 직접 전쟁에 나서는 순간, 인간의 전쟁 개념 자체가 무의미해짐
> - **에반과의 접점**: 에반이 숲에 들어서면 길이 열리고 나무가 화살을 막아준다 — 「숲이 하나의 군대」
